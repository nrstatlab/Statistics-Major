Experiment 2 — Skewness & Kurtosis from a PMF

Continuing from Experiment 1 with the same PMF:

\(\beta_1 = \mu_3^2 / \mu_2^3 = 0.0359 / 2.908 = 0.01235\).

\(\gamma_1 = \mu_3 / \mu_2^{3/2} = -0.1895 / 1.7053 = -0.111\) (slightly left-skewed).

\(\beta_2 = \mu_4 / \mu_2^2 = 4.412 / 2.0378 = 2.165\) (Platykurtic since β₂ < 3).

\(\gamma_2 = \beta_2 - 3 = -0.835\).

Experiment 3 — Moments from a PDF

PROBLEM

\(X\) has PDF \(f(x) = 6x(1-x),\; 0 \le x \le 1\) (Beta(2,2)). Compute first four raw moments.

Solution:

Experiment 5 — Joint, Marginal & Conditional PMF; Independence

PROBLEM

The joint PMF of \((X,Y)\) is:

X \ Y123
01/122/121/12
12/123/123/12

(a) Find marginal PMFs.
(b) Find \(P(Y=2 | X=1)\).
(c) Are \(X\) and \(Y\) independent?

(a) Marginals:

\(p_X(0) = 1/12 + 2/12 + 1/12 = 4/12 = 1/3\); \(p_X(1) = 2/12 + 3/12 + 3/12 = 8/12 = 2/3\).

\(p_Y(1) = 1/12 + 2/12 = 3/12 = 1/4\); \(p_Y(2) = 5/12\); \(p_Y(3) = 4/12 = 1/3\).

(b) Conditional: \(P(Y=2|X=1) = (3/12)/(8/12) = 3/8\).

(c) Independence check: \(p_X(0) p_Y(1) = (1/3)(1/4) = 1/12\), and \(p(0,1) = 1/12\) ✓.

But \(p_X(1) p_Y(3) = (2/3)(1/3) = 2/9 \ne 3/12 = 1/4\). Hence not independent.

Experiment 6 — Joint, Marginal & Conditional PDF; Independence

PROBLEM

\(f(x,y) = 4xy,\; 0 \le x,y \le 1\). Find marginals, conditional density, and check independence.

Marginals:

\(f_X(x) = \int_0^1 4xy\,dy = 4x \cdot 1/2 = 2x\) for \(0 \le x \le 1\).

\(f_Y(y) = 2y\) for \(0 \le y \le 1\) (by symmetry).

Conditional: \(f_{Y|X}(y|x) = (4xy)/(2x) = 2y\).

Since \(f_{Y|X}(y|x) = f_Y(y)\) and \(f(x,y) = f_X(x) f_Y(y) = 2x \cdot 2y = 4xy\) ✓ — \(X\) and \(Y\) are independent.

Second problem (Not independent)

PROBLEM

\(f(x,y) = x + y,\; 0 \le x,y \le 1\). Examine independence.

\(f_X(x) = x + 1/2;\; f_Y(y) = y + 1/2\). Product = \((x+1/2)(y+1/2) \ne x+y\). Hence not independent.

Experiment 7 — Chebyshev's Inequality Applications

PROBLEM 1

The mean and SD of marks of 1 000 students are 60 and 8. Find a lower bound for the number of students whose marks lie between 44 and 76.

Solution: The interval \([44, 76]\) is \(\mu \pm 2\sigma\) (since \(2 \times 8 = 16\)).

\(P(|X - 60| < 16) \ge 1 - 1/4 = 3/4 = 0.75\). So at least \(0.75 \times 1000 = \mathbf{750}\) students lie in this range.

PROBLEM 2

Determine sample size \(n\) so that the sample mean of i.i.d. variables (with \(\sigma^2 = 4\)) is within 0.5 of \(\mu\) with probability at least 0.95.

Solution: \(\bar X_n\) has variance \(\sigma^2/n = 4/n\). Apply Chebyshev:

\(P(|\bar X - \mu| \ge 0.5) \le \dfrac{4/n}{0.25} = \dfrac{16}{n}\). For this to be \(\le 0.05\): \(n \ge 320\).

Note: Chebyshev gives a conservative bound. Using a normal approximation (when justified) typically gives much smaller required \(n\).

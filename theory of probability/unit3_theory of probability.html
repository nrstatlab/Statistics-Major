Unit 3 — Bivariate Random Variables

Joint, marginal and conditional distributions; independence of random variables — both discrete and continuous cases.

Topics Covered

Bivariate r.v. Joint PMF (jpmf) Joint PDF (jpdf) Marginal Distributions Conditional Distributions Independence Joint CDF

1. Bivariate Random Variable

DEFINITION

If two random variables \(X\) and \(Y\) are defined on the same sample space, the pair \((X,Y)\) is called a bivariate (two-dimensional) random variable. It can be discrete or continuous.

EXAMPLE 1

Two coins are tossed. \(X\) = number of heads on first coin, \(Y\) = total number of heads. \((X,Y)\) takes values (0,0),(0,1),(1,1),(1,2) — a bivariate discrete r.v.

EXAMPLE 2

\(X\) = height (cm), \(Y\) = weight (kg) of a randomly chosen person. Both are continuous, so \((X,Y)\) is a bivariate continuous r.v.

2. Joint Probability Mass Function (jpmf)

DEFINITION

For a discrete bivariate r.v. \((X,Y)\), the joint PMF is

\[ p(x,y) \;=\; P(X = x,\; Y = y). \]
PROPERTIES \[ (i)\;\; p(x,y) \ge 0; \qquad (ii)\;\; \sum_{x}\sum_{y} p(x,y) \;=\; 1 \]
EXAMPLE 1

Given joint PMF table:

X \ Y123Row sum (Marginal of X)
00.100.200.100.40
10.200.300.100.60
Col sum (Marg. Y)0.300.500.201.00

All entries ≥ 0 and total = 1; valid joint PMF.

EXAMPLE 2 (Find a constant)

\(p(x,y) = k(x+y),\; x = 1,2;\; y = 1,2,3.\)
\(\sum p = k[(1+1)+(1+2)+(1+3)+(2+1)+(2+2)+(2+3)] = k(2+3+4+3+4+5) = 21k = 1 \Rightarrow k = 1/21\).

3. Joint Probability Density Function (jpdf)

DEFINITION

For continuous \((X,Y)\), the joint PDF \(f(x,y)\) satisfies:

\[ (i)\;\; f(x,y) \ge 0;\qquad (ii)\;\; \int_{-\infty}^{\infty}\!\!\int_{-\infty}^{\infty} f(x,y)\,dx\,dy = 1;\qquad (iii)\;\; P((X,Y) \in A) = \iint_A f(x,y)\,dx\,dy \]
EXAMPLE 1 (Find constant)

\(f(x,y) = k(x+y),\; 0 \le x,y \le 1\).
\(\int_0^1 \int_0^1 k(x+y) dx dy = k \int_0^1 (1/2 + y) dy = k(1/2 + 1/2) = k = 1\).
Hence \(k = 1\) — the function \(f(x,y) = x+y\) on the unit square.

EXAMPLE 2 (Probability over a region)

For \(f(x,y) = 4xy,\; 0 \le x \le 1,\; 0 \le y \le 1\):

\(P(X \le 1/2,\; Y \le 1/2) = \int_0^{1/2}\int_0^{1/2} 4xy\,dy\,dx = (1/4)(1/4)\cdot \) [check] \(= \int_0^{1/2} 4x \cdot (1/8) dx = (1/2) \int_0^{1/2} x dx = (1/2)(1/8) = 1/16\).

4. Marginal Distributions

DEFINITION

The marginal distribution of one variable is obtained by summing (or integrating) the joint distribution over the other variable.

Discrete (Marginal PMF — mpmf)

\[ p_X(x) \;=\; \sum_{y} p(x,y), \qquad p_Y(y) \;=\; \sum_{x} p(x,y) \]

Continuous (Marginal PDF — mpdf)

\[ f_X(x) \;=\; \int_{-\infty}^{\infty} f(x,y)\,dy, \qquad f_Y(y) \;=\; \int_{-\infty}^{\infty} f(x,y)\,dx \]
EXAMPLE 1 (Discrete)

From Example 1 of Section 2: marginal of \(X\) is \(p_X(0) = 0.40,\; p_X(1) = 0.60\). Marginal of \(Y\) is \(p_Y(1) = 0.30,\; p_Y(2) = 0.50,\; p_Y(3) = 0.20\).

EXAMPLE 2 (Continuous)

For \(f(x,y) = x+y,\; 0 \le x,y \le 1\):

\(f_X(x) = \int_0^1 (x+y) dy = x + 1/2\) for \(0 \le x \le 1\).

\(f_Y(y) = \int_0^1 (x+y) dx = 1/2 + y\) for \(0 \le y \le 1\).

5. Conditional Distributions

DEFINITION

The conditional distribution of \(Y\) given \(X = x\) describes the probability of \(Y\) when \(X\) takes a fixed value.

Discrete (Conditional PMF)

\[ p_{Y|X}(y \mid x) \;=\; \dfrac{p(x,y)}{p_X(x)}, \quad p_X(x) > 0 \]

Continuous (Conditional PDF)

\[ f_{Y|X}(y \mid x) \;=\; \dfrac{f(x,y)}{f_X(x)}, \quad f_X(x) > 0 \]
EXAMPLE 1 (Discrete)

From Section 2 Example 1, find conditional distribution of \(Y\) given \(X = 1\).

\(p_X(1) = 0.60\); \(p_{Y|X}(1|1) = 0.20/0.60 = 1/3,\; p_{Y|X}(2|1) = 0.30/0.60 = 1/2,\; p_{Y|X}(3|1) = 0.10/0.60 = 1/6\).

(They sum to 1 ✓.)

EXAMPLE 2 (Continuous)

For \(f(x,y) = x+y\) on the unit square: \(f_X(x) = x + 1/2\).

\(f_{Y|X}(y|x) = (x+y)/(x+1/2)\) for \(0 \le y \le 1\).

Verify: \(\int_0^1 (x+y)/(x+1/2)\,dy = (x + 1/2)/(x+1/2) = 1\) ✓.

6. Independence of Random Variables

DEFINITION

\(X\) and \(Y\) are independent if their joint distribution is the product of marginals:

Discrete: \(p(x,y) = p_X(x) \cdot p_Y(y)\) for all \(x,y\).

Continuous: \(f(x,y) = f_X(x) \cdot f_Y(y)\) for all \(x,y\).

Equivalently, the conditional equals the marginal: \(f_{Y|X}(y|x) = f_Y(y)\).

EXAMPLE 1 (Independent)
X \ Y12pX

pY0.400.601.00

Check: \(p(0,1) = 0.12 = 0.30 \times 0.40\); \(p(1,2) = 0.42 = 0.70 \times 0.60\). Yes — independent.

EXAMPLE 2 (Not independent)

For \(f(x,y) = x + y\) on unit square: \(f_X(x) = x + 1/2\), \(f_Y(y) = y + 1/2\).

Product = \((x+1/2)(y+1/2)\) which is not equal to \(x+y\). Hence \(X,Y\) are not independent.

7. Joint Cumulative Distribution Function

\[ F(x,y) \;=\; P(X \le x,\; Y \le y) \]

Properties

  • \(F(-\infty, y) = F(x, -\infty) = 0;\; F(\infty,\infty) = 1\).
  • Non-decreasing in each argument.
  • For continuous \((X,Y)\): \(\dfrac{\partial^2 F}{\partial x\, \partial y} = f(x,y)\).
  • \(P(a_1 < X \le a_2,\; b_1 < Y \le b_2) = F(a_2,b_2) - F(a_1,b_2) - F(a_2,b_1) + F(a_1,b_1)\).
EXAMPLE 1

For \(f(x,y) = 1\) on unit square: \(F(x,y) = xy\) for \(0 \le x,y \le 1\).

EXAMPLE 2

For \(f(x,y) = e^{-(x+y)},\; x,y \ge 0\):

\(F(x,y) = (1-e^{-x})(1-e^{-y})\). Marginals: each is Exp(1); they are independent (product form).

Key Take-aways from Unit 3

  • jpmf / jpdf: probability that two r.v.'s take values jointly.
  • Marginal = sum/integrate the joint over the other variable.
  • Conditional = joint / marginal.
  • \(X \perp Y\) (independent) iff joint = product of marginals (or conditional = marginal).
  • Joint CDF: \(F(x,y) = P(X \le x, Y \le y)\); its mixed partial derivative gives the density.

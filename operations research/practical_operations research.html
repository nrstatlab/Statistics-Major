Experiment 1 — Graphical Method

PROBLEM

Max \(Z = 5 x_1 + 4 x_2\) subject to

  • \(6 x_1 + 4 x_2 \le 24\)
  • \(x_1 + 2 x_2 \le 6\)
  • \(-x_1 + x_2 \le 1\)
  • \(x_2 \le 2\)
  • \(x_1, x_2 \ge 0\)

Solution:

  1. Plot constraints and identify the feasible polygon.
  2. Vertices: \(O(0,0),\; (4,0),\; (3,1.5),\; (2,2),\; (1,2),\; (0,1)\).
  3. Evaluate Z at each:
VertexZ = 5x₁ + 4x₂
(0, 0)0
(4, 0)20
(3, 1.5)21 (max)
(2, 2)18
(1, 2)13
(0, 1)4

Result: \(x_1 = 3,\; x_2 = 1.5,\; Z_{\max} = 21\).

Experiment 2 — Simplex Method

PROBLEM

Max \(Z = 3 x_1 + 2 x_2\) subject to \(x_1 + x_2 \le 4,\; x_1 + 3 x_2 \le 6,\; x_1, x_2 \ge 0\).

Steps

  1. Add slacks \(s_1, s_2\); initial BFS: \((x_1, x_2, s_1, s_2) = (0, 0, 4, 6)\).
  2. Pivot in \(x_1\) (largest \(c_j - z_j\)); leaving \(s_1\); new BFS \((4, 0, 0, 2)\).
  3. All \(c_j - z_j \le 0\) ⇒ optimum.

Result: \(x_1 = 4,\; x_2 = 0,\; Z_{\max} = 12\).

Verification by graphical method

Plot constraints: vertices (0,0), (4,0), (3,1), (0,2). Z values: 0, 12, 11, 4. Max = 12 ✓.

Experiment 3 — Big-M Method

PROBLEM

Min \(Z = 7 x_1 + 9 x_2\) subject to \(3 x_1 + 6 x_2 \ge 18,\; 8 x_1 + 4 x_2 \ge 16,\; x_1, x_2 \ge 0\).

Setup

\[ \begin{aligned} \text{Min}\;Z &= 7x_1 + 9x_2 + 0 s_1 + 0 s_2 + M A_1 + M A_2 \\ 3 x_1 + 6 x_2 - s_1 + A_1 &= 18 \\ 8 x_1 + 4 x_2 - s_2 + A_2 &= 16 \end{aligned} \]

Iterations proceed as in Unit 4. After two iterations, both artificials leave the basis and the optimum is reached.

Result: \(x_1 = 4/5 = 0.8,\; x_2 = 14/5 = 2.8,\; Z_{\min} = 7(0.8) + 9(2.8) = 30.8\) (approximately).

Experiment 4 — Two-Phase Method

PROBLEM

Same as Experiment 3 — solve using Two-Phase method.

Phase 1

Minimise \(W = A_1 + A_2\) subject to the constraints (with surplus + artificial). After two iterations, \(W = 0\) and a BFS \((x_1, x_2) = (0.8, 2.8)\) is found.

Phase 2

Drop artificials. Use original objective Min \(Z = 7 x_1 + 9 x_2\). At the BFS, check optimality: \(c_j - z_j \ge 0\) ⇒ optimal.

Result: Same as Experiment 3 — \(x_1 = 0.8, x_2 = 2.8, Z_{\min} \approx 30.8\). Two-Phase and Big-M produce identical optima.

Experiment 5 — Special Cases

5.1 Unbounded Solution

PROBLEM

Max \(Z = 3 x_1 + 5 x_2\) subject to \(x_1 + x_2 \ge 2,\; x_1 - x_2 \le 1,\; x_1, x_2 \ge 0\).

Apply simplex (with artificials for ≥). At some iteration, the entering column has all entries ≤ 0 ⇒ unbounded. Conclude: \(Z \to \infty\).

5.2 Alternative Optima

PROBLEM

Max \(Z = 2 x_1 + 4 x_2\) subject to \(x_1 + 2 x_2 \le 5,\; x_1 + x_2 \le 4,\; x_1, x_2 \ge 0\).

Solve by simplex. At optimum, a non-basic variable has \(c_j - z_j = 0\) ⇒ multiple optima.

Two optimal vertices: \((0, 2.5)\) with \(Z = 10\) and \((3, 1)\) with \(Z = 10\). Any convex combination is optimal.

Experiment 6 — Principle of Duality

PROBLEM

Write the dual of and solve via primal:

Max \(Z = 4 x_1 + 5 x_2\) s.t. \(2 x_1 + x_2 \le 8,\; x_1 + 2 x_2 \le 10,\; x_1, x_2 \ge 0\).

Dual

Min \(W = 8 y_1 + 10 y_2\) s.t. \(2 y_1 + y_2 \ge 4,\; y_1 + 2 y_2 \ge 5,\; y_1, y_2 \ge 0\).

Solution

Solve primal directly by simplex: \(x_1 = 2, x_2 = 4, Z^{*} = 28\).

Read shadow prices from primal's final tableau (under slack columns): \(y_1^{*} = 1, y_2^{*} = 2\).

Check duality: \(W^{*} = 8(1) + 10(2) = 28 = Z^{*}\) ✓ (strong duality).

Verify complementary slackness:

  • Both primal constraints are binding (slack = 0) ⇒ both dual variables positive. ✓
  • Both primal variables are positive ⇒ both dual constraints are binding. Check: \(2(1) + 2 = 4\) ✓; \(1 + 2(2) = 5\) ✓.

Experiment 7 — Dual Simplex Method

PROBLEM

Min \(Z = 2 x_1 + x_2\) subject to \(3 x_1 + x_2 \ge 3,\; 4 x_1 + 3 x_2 \ge 6,\; x_1 + 2 x_2 \le 3,\; x_1, x_2 \ge 0\).

Setup for Dual Simplex

Convert ≥ to ≤ by multiplying by −1; convert Min to Max by negating Z.

\[ \begin{aligned} \text{Max}\;Z' &= -2 x_1 - x_2 \\ -3 x_1 - x_2 + s_1 &= -3 \\ -4 x_1 - 3 x_2 + s_2 &= -6 \\ x_1 + 2 x_2 + s_3 &= 3 \end{aligned} \]

Initial Tableau

Dual feasible (all \(c_j - z_j \le 0\)), primal infeasible (negative b's). Apply dual simplex:

  • Most negative b: \(-6\) ⇒ row 2 (s₂) leaves.
  • Negative entries in row 2: \(-4\) (x₁), \(-3\) (x₂). Compute ratios:
    • x₁: \(-2 / -4 = 0.5\)
    • x₂: \(-1 / -3 = 0.33\)
    Smallest ratio is for x₂ ⇒ x₂ enters. Pivot = −3.
  • Iterate further if needed.

Final result: After completing the iterations, \(x_1 = 3/5,\; x_2 = 6/5,\; Z_{\min} = 12/5 = 2.4\).

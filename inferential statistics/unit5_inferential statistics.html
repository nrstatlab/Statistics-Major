Unit 5 — Non-parametric Tests

Distribution-free tests: Run test, Sign test, Wilcoxon signed-rank, Median test, Mann–Whitney U test, Wald–Wolfowitz runs test.

Topics Covered

1. Introduction

DEFINITION

Non-parametric (distribution-free) tests are statistical tests that do not assume any particular form for the underlying population distribution. They typically work with ranks or signs rather than the actual data values.

Advantages

  1. Make minimal assumptions — useful when distribution is unknown or non-normal.
  2. Applicable to ordinal data (ranks).
  3. Robust to outliers.
  4. Easier to compute and explain.

Disadvantages

  1. Less powerful than parametric tests when the parametric assumptions are met.
  2. Use less information from the data (ranks instead of actual values).
  3. Often not as suitable for complex models.
  4. Tables / exact distributions required for small samples.

2. One-Sample Runs Test (Test for Randomness)

PURPOSE

Tests whether a sequence of two types of items (or values above/below the median) is random.

Procedure

  1. Mark each observation as type 1 or type 2 (above/below median).
  2. Count the total runs \(R\) (maximal blocks of same type).
  3. Let \(n_1, n_2\) be counts of each type.
  4. For \(n_1, n_2 > 10\), use normal approximation: \[ E(R) = \dfrac{2 n_1 n_2}{n_1 + n_2} + 1, \quad \text{Var}(R) = \dfrac{2 n_1 n_2 (2 n_1 n_2 - n_1 - n_2)}{(n_1 + n_2)^2 (n_1 + n_2 - 1)}. \] \[ Z = \dfrac{R - E(R)}{\sqrt{\text{Var}(R)}} \sim N(0, 1). \]
  5. Too few runs ⇒ clustering; too many runs ⇒ alternation.
EXAMPLE 1

Sequence: H T H T H T H T H T H T (12 tosses) — \(n_1 = n_2 = 6\), \(R = 12\) (alternating). \(E(R) = 7,\; \text{Var}(R) \approx 2.91,\; Z = 5/1.71 = 2.93\) ⇒ reject randomness.

EXAMPLE 2

Sequence: H H H H H H T T T T T T — \(R = 2,\; Z = (2-7)/1.71 = -2.93\) ⇒ also reject (clustering, not random).

3. Sign Test

Tests \(H_0\): median = \(M_0\) using only the signs of \((X_i - M_0)\). Or for paired samples, signs of \((X_i - Y_i)\).

Procedure

  1. Discard observations equal to \(M_0\) (or paired ties).
  2. Let \(n\) = total number of non-zero signs; \(S\) = number of "+" signs.
  3. Under \(H_0\), \(S \sim \text{Binomial}(n, 0.5)\).
  4. For large \(n\), use normal approximation: \(Z = (S - n/2)/\sqrt{n/4}\).
EXAMPLE 1 (Single sample)

15 measurements; 4 are below 50 (−), 11 above (+). \(n = 15, S = 11\). Two-tailed P-value from Binomial(15, 0.5): \(2 P(S \ge 11) = 2(0.0592) = 0.118\) ⇒ accept \(H_0\) at 5 %.

EXAMPLE 2 (Paired)

Before-after for 12 patients: +9 −2 (1 tie). \(n = 11, S_+ = 9\). \(P(S_+ \ge 9) = 0.0327\); two-tailed p = 0.065 ⇒ accept \(H_0\) at 5 %.

4. Wilcoxon Signed-Rank Test

More powerful than the sign test because it uses the magnitudes of differences as well.

Procedure

  1. Compute differences \(d_i = X_i - M_0\) (or paired \(X_i - Y_i\)). Discard zeros.
  2. Rank \(|d_i|\) (smallest = 1).
  3. Attach the sign of \(d_i\) to each rank.
  4. Let \(W^+\) = sum of positive ranks; \(W^-\) = sum of negative ranks. Test statistic \(W = \min(W^+, W^-)\).
  5. For small \(n\), use Wilcoxon tables; for large \(n\): \[ Z = \dfrac{W - n(n+1)/4}{\sqrt{n(n+1)(2n+1)/24}} \sim N(0,1). \]
EXAMPLE 1

Differences \(d\): −2, +5, +1, −3, +6, +4. \(|d|\): 1, 2, 3, 4, 5, 6 ranked 1, 2, 3, 4, 5, 6. With signs: +1, −2, −3, +4, +5, +6 (matching order). \(W^+ = 1+4+5+6 = 16,\; W^- = 2+3 = 5\). \(W = 5\). For \(n = 6\), critical at 5 % two-tailed = 0; \(W = 5 > 0\) ⇒ accept \(H_0\).

EXAMPLE 2

10 paired differences with \(W^+ = 50, W^- = 5\). \(W = 5\). Critical Wilcoxon for \(n = 10\) at 5 % two-tailed = 8; \(W < 8\) ⇒ reject \(H_0\) — significant difference.

5. Median Test (Two Independent Samples)

Tests whether two samples come from populations with the same median.

Procedure

  1. Combine both samples and find the grand median \(M\).
  2. Form a 2 × 2 contingency table: counts of values above/below \(M\) for each sample.
  3. Apply the χ² test of independence.
EXAMPLE 1

Sample A (10 obs): 4 above, 6 below grand median. Sample B (10 obs): 6 above, 4 below.

Above MBelow M
A46
B64

Expected = 5 each. \(\chi^2 = 4(1)/5 = 0.80\). df = 1, \(\chi^2_{0.05} = 3.84\) ⇒ accept \(H_0\).

EXAMPLE 2

If Sample A has 9 above / 1 below and Sample B has 1 above / 9 below: \(\chi^2 = 4(4)^2/5 = 12.8\) ⇒ reject \(H_0\); medians differ.

6. Wilcoxon–Mann–Whitney U Test

The most powerful non-parametric test for comparing two independent samples. Tests whether the distributions are identical (or whether one tends to produce larger values).

Procedure

  1. Combine both samples and rank from smallest (1) to largest.
  2. Let \(R_1\) = sum of ranks of sample 1; \(R_2\) = sum of ranks of sample 2. Note \(R_1 + R_2 = N(N+1)/2\) where \(N = n_1 + n_2\).
  3. Compute U statistics: \[ U_1 = R_1 - \dfrac{n_1(n_1 + 1)}{2}, \quad U_2 = R_2 - \dfrac{n_2(n_2 + 1)}{2}. \]
  4. Test statistic \(U = \min(U_1, U_2)\). Compare with table; or for large \(n\): \[ Z = \dfrac{U - n_1 n_2 / 2}{\sqrt{n_1 n_2 (n_1 + n_2 + 1)/12}}. \]
EXAMPLE 1

Sample 1 (n₁=4): 12, 15, 9, 18. Sample 2 (n₂=5): 7, 10, 14, 11, 13.

Combined ranks: 7→1, 9→2, 10→3, 11→4, 12→5, 13→6, 14→7, 15→8, 18→9.

\(R_1 = 2 + 5 + 8 + 9 = 24,\; R_2 = 1+3+4+6+7 = 21\).

\(U_1 = 24 - 4 \cdot 5/2 = 14,\; U_2 = 21 - 5 \cdot 6/2 = 6\). \(U = 6\). Critical for (4,5) at 5 % two-tailed = 1; \(U = 6 > 1\) ⇒ accept \(H_0\).

EXAMPLE 2

Two methods scored over n₁ = 12, n₂ = 15 with \(R_1 = 250\). \(U_1 = 250 - 12(13)/2 = 172\). \(Z = (172 - 90)/\sqrt{12 \cdot 15 \cdot 28/12} = 82/14.5 = 5.66\) ⇒ reject \(H_0\); methods differ.

7. Wald–Wolfowitz Runs Test (Two Samples)

Tests whether two independent samples come from the same population. Uses runs in the combined ranked sample.

Procedure

  1. Combine the two samples and arrange in ascending order.
  2. Replace each observation by its sample label (A or B).
  3. Count the number of runs \(R\).
  4. Few runs ⇒ samples differ (each clusters separately).
  5. For large \(n_1, n_2\), use the normal approximation (same formulas as one-sample runs test).
EXAMPLE 1

Sample A: 1,2,3,4,5; Sample B: 6,7,8,9,10. Combined ordered: A A A A A B B B B B → \(R = 2\). \(E(R) = 6, \text{Var}(R) = 2.22\). \(Z = (2-6)/1.49 = -2.68\) ⇒ reject \(H_0\); distributions differ.

EXAMPLE 2

Sample A: 1, 4, 6, 9; Sample B: 2, 3, 5, 7, 8, 10. Order: A B B A B A B B A B → \(R = 8\). \(n_1=4, n_2=6, E(R)=5.8, \text{Var}(R)=1.96\). \(Z = (8-5.8)/1.4 = 1.57\) ⇒ accept \(H_0\) at 5 %.

Key Take-aways

  • Non-parametric tests are distribution-free; trade some power for robustness.
  • Use ranks/signs/counts instead of original data.
  • Sign test is simplest; Wilcoxon signed-rank uses magnitudes too — more powerful.
  • Mann–Whitney U is the rank-based two-sample t analogue.
  • Runs tests detect non-randomness or distribution differences via runs.

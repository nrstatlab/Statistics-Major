Unit 2 — Testing of Hypothesis

Statistical hypotheses, critical region, two types of errors, level of significance, p-value, power, one- & two-tailed tests, Neyman–Pearson lemma.

Topics Covered

Null & Alternative Critical Region Type I & II Errors Significance Level p-value Power of a Test One- vs Two-tailed Neyman–Pearson

1. Statistical Hypothesis

DEFINITION

A statistical hypothesis is a statement about the value of one or more parameters of a population. We then use sample data to test whether this statement is consistent with the evidence.

Null and Alternative Hypotheses

  • Null hypothesis \(H_0\) — the assertion to be tested; usually a "no-effect" / "no-difference" claim.
  • Alternative hypothesis \(H_1\) (or \(H_a\)) — what we accept if \(H_0\) is rejected.

Possible forms of \(H_1\):

FormTypeExample
\(H_1: \theta \ne \theta_0\)Two-tailedμ ≠ 50
\(H_1: \theta > \theta_0\)Right-tailedμ > 50
\(H_1: \theta < \theta_0\)Left-tailedμ < 50

Simple vs Composite hypothesis:

  • Simple — completely specifies the distribution (e.g., \(H_0: \mu = 50, \sigma = 5\)).
  • Composite — does not (e.g., \(H_0: \mu = 50\) with \(\sigma\) unknown, or \(H_1: \mu > 50\)).

2. Critical Region (Rejection Region)

The critical region \(W\) is the set of sample-space outcomes for which we reject \(H_0\). Its complement is the acceptance region.

For a continuous test statistic \(T\) with a critical value \(c\):

  • Right-tailed: reject \(H_0\) if \(T > c\).
  • Left-tailed: reject \(H_0\) if \(T < c\).
  • Two-tailed: reject \(H_0\) if \(|T| > c\).

3. Two Types of Errors

H₀ TrueH₀ False
Reject H₀Type I error (probability α)Correct decision (probability 1−β)
Accept H₀Correct decision (probability 1−α)Type II error (probability β)
  • Level of significance (α) = P(Type I error) = P(reject \(H_0\) | \(H_0\) true). Common α: 0.05, 0.01.
  • β = P(Type II error) = P(accept \(H_0\) | \(H_1\) true).
  • Power = 1 − β = P(reject \(H_0\) | \(H_1\) true) = ability to detect a true effect.

Reducing α typically increases β and vice versa. Increasing sample size \(n\) is the proper way to reduce both.

EXAMPLE 1

A factory's quality engineer rejects a defect-free batch (Type I error) — money lost on rework. With α = 0.05, this happens 5 % of the time even though the batch was fine.

EXAMPLE 2

A medical test fails to detect a sick patient (Type II error). If β = 0.20, only 80 % of truly ill patients are correctly identified — power = 0.80.

4. p-value

DEFINITION

The p-value is the probability, assuming \(H_0\) is true, of observing a test statistic as extreme as (or more extreme than) the one observed.

Decision Rule

  • If p-value < α: reject \(H_0\).
  • If p-value ≥ α: do not reject \(H_0\).
  • Smaller p-value ⇒ stronger evidence against \(H_0\).
EXAMPLE 1

For an observed Z = 2.10 in a two-tailed test, p = 2(1 − Φ(2.10)) = 2(0.0179) = 0.0358. Since p < 0.05, reject \(H_0\) at 5 %.

EXAMPLE 2

p = 0.18 in any test ⇒ data are consistent with \(H_0\); no significant evidence to reject.

5. Power of a Test

Power = \(1 - \beta\) = P(reject \(H_0\) | \(H_1\) true) — the probability of correctly detecting an effect.

Factors that Increase Power

  1. Larger sample size \(n\).
  2. Larger effect size (true value of \(\theta\) far from \(\theta_0\)).
  3. Larger α (less stringent test).
  4. Smaller population variance.
  5. One-tailed instead of two-tailed (when direction is known).

Power function \(\pi(\theta) = P(\text{reject } H_0 \mid \theta)\) is a function of the true \(\theta\). At \(\theta = \theta_0\), \(\pi = \alpha\); for \(\theta\) far from \(\theta_0\), \(\pi \to 1\).

7. Neyman–Pearson Lemma

STATEMENT

For testing simple \(H_0: \theta = \theta_0\) against simple \(H_1: \theta = \theta_1\), the test that rejects \(H_0\) when

\[ \dfrac{L(\theta_1; \mathbf x)}{L(\theta_0; \mathbf x)} \;\ge\; k, \]

(where \(k\) is chosen so the size is \(\alpha\)) is the most powerful test (MPT) of size \(\alpha\). The test statistic is the likelihood ratio.

This lemma is the foundation of optimal hypothesis testing. For composite hypotheses, generalised likelihood ratio tests (GLRT) extend this idea.

Examples in Standard Distributions

(a) Binomial

\(H_0: p = p_0\) vs \(H_1: p = p_1 > p_0\). The MPT rejects \(H_0\) for large values of \(\sum X_i\).

(b) Poisson

\(H_0: \lambda = \lambda_0\) vs \(H_1: \lambda = \lambda_1 > \lambda_0\). MPT rejects for large \(\sum X_i\).

(c) Exponential

\(H_0: \lambda = \lambda_0\) vs \(H_1: \lambda = \lambda_1 < \lambda_0\). MPT based on \(\sum X_i\).

(d) Normal mean (σ known)

\(H_0: \mu = \mu_0\) vs \(H_1: \mu = \mu_1 > \mu_0\). MPT rejects when \(\bar X > c\), where \(c = \mu_0 + z_\alpha\sigma/\sqrt n\).

EXAMPLE 1 (Normal)

Test \(H_0: \mu = 100\) vs \(H_1: \mu = 105\) at α = 0.05 with σ = 10, n = 25.

Reject when \(\bar X > 100 + 1.645(10/5) = 103.29\). Power at \(\mu_1 = 105\): \(P(\bar X > 103.29 \mid \mu = 105) = P(Z > -0.855) = 0.804\).

EXAMPLE 2 (Binomial)

Test \(H_0: p = 0.5\) vs \(H_1: p = 0.6\) with n = 100. MPT rejects when \(X \ge c\). For α ≈ 0.05, \(c = 58\) (using normal approx). Power at \(p_1 = 0.6\) ≈ 0.69.

General Procedure for Hypothesis Testing

  1. State \(H_0\) and \(H_1\).
  2. Choose level of significance α.
  3. Identify the appropriate test statistic and its distribution under \(H_0\).
  4. Determine the critical region (or compute p-value).
  5. Compute the test statistic from data.
  6. Compare and conclude.

Key Take-aways

  • \(H_0\) (no effect) vs \(H_1\) (effect); only \(H_0\) is tested directly.
  • Type I (α) and Type II (β) errors trade off; power = 1 − β.
  • Critical region depends on form of \(H_1\): one-tailed or two-tailed.
  • p-value < α ⇔ reject \(H_0\).
  • Neyman–Pearson lemma → likelihood ratio test is most powerful for simple-vs-simple hypotheses.

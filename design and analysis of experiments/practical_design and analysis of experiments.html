PROBLEM

3 varieties: V1 (3): 20, 24, 22; V2 (4): 28, 30, 32, 34; V3 (2): 18, 20.

SST = 272, SSTr = 242, SSE = 30. df = (2, 6). F = 24.2 > 5.14 ⇒ reject \(H_0\).

Experiment 4 — Analysis of CRD & Critical Differences

PROBLEM

Same data as Experiment 1 (3 fertilizers × 4 reps). Compute CD and decide which pairs differ.

MSE = 5.33, dfE = 9, \(t_{0.025, 9} = 2.262\). CD = \(2.262\sqrt{2(5.33)/4} = 3.69\).

Means: \(\bar y_1 = 25, \bar y_2 = 33, \bar y_3 = 22\). Differences: |25 − 33| = 8 (differ); |25 − 22| = 3 (≤ CD ⇒ no diff); |33 − 22| = 11 (differ).

Experiment 5 — RBD Analysis & RE vs CRD

PROBLEM

3 treatments × 4 blocks. SSTr = 60, SSBl = 24, SSE = 12, dfE = 6, MSE = 2.

FTr = (60/2)/2 = 15 ⇒ significant. CD = \(t_{0.025,6}\sqrt{2 \cdot 2/4} = 2.447 \cdot 1 = 2.447\).

RE of RBD over CRD:

MSECRD = (24 + 12)/(3 + 6) = 36/9 = 4. RE = 4/2 = 2.0 → RBD twice as efficient.

Experiment 6 — Missing Value in RBD

PROBLEM

3 treatments × 3 blocks. Value at T2-B3 missing.

T381012

\(T'_2 = 30, B'_3 = 26, G' = 96, t = b = 3\).

\(\hat y = [3(30) + 3(26) - 96]/[(2)(2)] = (90 + 78 - 96)/4 = 72/4 = 18\).

Insert 18 and proceed with RBD analysis; reduce error df by 1.

Experiment 7 — LSD Analysis & Efficiency vs CRD/RBD

PROBLEM

5 × 5 LSD with SSR = 30, SSC = 25, SSTr = 200, SSE = 60, dfE = 12.

MSELSD = 5. FTr = (200/4)/5 = 10 > 3.26 ⇒ treatments differ.

CD = \(t_{0.025, 12}\sqrt{2 \cdot 5/5} = 2.179\sqrt{2} = 3.08\).

RE of LSD over CRD: MSECRD = (30 + 25 + 60)/20 = 5.75. RE = 5.75/5 = 1.15 (15 % gain).

RE of LSD over RBD (cols as blocks): MSERBD = (30 + 60)/16 = 5.625. RE = 5.625/5 = 1.125 (12.5 % gain).

RE of LSD over RBD (rows as blocks): MSERBD = (25 + 60)/16 = 5.31. RE = 5.31/5 = 1.06 (6 % gain).

Experiment 8 — Missing Value in LSD

PROBLEM

4 × 4 LSD: row total \(R' = 65\), column total \(C' = 70\), treatment total \(T' = 60\), grand total \(G' = 240\), \(t = 4\).

\(\hat y = [4(65 + 70 + 60) - 2(240)] / [(3)(2)] = (780 - 480)/6 = 50\).

Insert 50 and analyse LSD; reduce error df by 1.

Lab Notebook Format

  1. Experiment number and title.
  2. Aim and design used (CRD/RBD/LSD).
  3. Data table.
  4. Total sums of squares and partition.
  5. ANOVA table with F values.
  6. Conclusion (with critical differences if treatment effect is significant).
  7. Relative efficiency where required.

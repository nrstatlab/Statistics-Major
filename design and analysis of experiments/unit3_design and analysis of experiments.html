Unit 3 — Randomised Block Design (RBD)

Concept, layout, advantages and applications; statistical analysis with worked examples; critical differences; missing-value estimation in RBD.

Topics Covered

1. Concept of RBD

DEFINITION

Randomised Block Design (RBD) uses all three Fisher principles — replication, randomization and local control. The experimental units are grouped into homogeneous blocks; within each block, treatments are randomly allotted, with each treatment appearing exactly once per block.

RBD is appropriate when there is a known source of variability across the experimental area (gradient of soil fertility, batch of animals, time periods). Blocking removes that variability from the error.

2. Layout of RBD

For \(t\) treatments and \(b\) blocks, total plots \(= t \times b\). Within each block of \(t\) plots, the \(t\) treatments are randomly assigned.

3. Advantages and Disadvantages

Advantages

  1. More precise than CRD when block-to-block variation is significant.
  2. Simple statistical analysis (two-way ANOVA without interaction).
  3. Flexible: any number of treatments and blocks.
  4. Missing observations can still be recovered via formula.

Disadvantages

  1. Requires homogeneity within blocks; if violated, RBD loses efficiency.
  2. If number of treatments is large, blocks become large and homogeneity is harder.
  3. Less flexible than CRD when replicates differ across treatments.

Applications

  • Agricultural field trials with a fertility gradient.
  • Industrial experiments where machines or operators cause variation.
  • Animal feeding trials where animals can be paired by weight/age.

4. Statistical Analysis of RBD

Mathematical Model

\[ y_{ij} = \mu + \alpha_i + \beta_j + \epsilon_{ij}, \]

where \(\alpha_i\) = effect of treatment \(i\), \(\beta_j\) = effect of block \(j\), \(\epsilon_{ij} \sim N(0,\sigma^2)\) and \(\sum \alpha_i = \sum \beta_j = 0\).

Hypotheses

  • \(H_0^T: \alpha_1 = \cdots = \alpha_t = 0\) (no treatment effect)
  • \(H_0^B: \beta_1 = \cdots = \beta_b = 0\) (no block effect)

Sums of Squares

Notation: \(T_i\) treatment totals (\(b\) obs each); \(B_j\) block totals (\(t\) obs each); \(G\) grand total; \(N = t b\); \(C = G^2/N\).

\[ \text{SS}_T = \sum y_{ij}^2 - C, \quad \text{SS}_{Tr} = \dfrac{\sum T_i^2}{b} - C, \quad \text{SS}_{Bl} = \dfrac{\sum B_j^2}{t} - C, \] \[ \text{SS}_E = \text{SS}_T - \text{SS}_{Tr} - \text{SS}_{Bl}. \]

Critical Difference

\[ \text{CD} \;=\; t_{\alpha/2,\,(t-1)(b-1)} \,\sqrt{\dfrac{2\, \text{MS}_E}{b}}. \]

Used to compare any pair of treatment means: differ if |\(\bar y_i - \bar y_j\)| > CD.

5. Estimating One Missing Value in RBD

If one observation \(y_{ij}\) is missing (in row \(i\), column \(j\)), it can be estimated by minimising the error sum of squares.

YATES' FORMULA (Missing value in RBD) \[ \hat y_{ij} \;=\; \dfrac{t\, T'_i + b\, B'_j - G'}{(t - 1)(b - 1)}, \]

where \(T'_i, B'_j\) are the sums of available values in the corresponding treatment row and block column; \(G'\) is the grand total of all available values; \(t\) = number of treatments; \(b\) = number of blocks.

Adjustments after substitution

  1. Insert \(\hat y_{ij}\) and analyse as usual.
  2. Reduce error df by 1 (loss of one observation): dfE = \((t-1)(b-1) - 1\).
  3. Treatment SS is biased upward — apply the bias correction: \[ B = \dfrac{(B'_j - (t-1)\hat y_{ij})^2}{t(t-1)}. \] Reduce SSTr by \(B\) before testing treatments.

\(\hat y_{ij} = \dfrac{4(80) + 5(60) - 380}{(3)(4)} = \dfrac{320 + 300 - 380}{12} = \dfrac{240}{12} = 20\).

Key Take-aways

  • RBD applies all three principles — replication, randomization, local control.
  • Layout: each treatment appears once per block; randomized within block.
  • Analysis = two-way ANOVA without interaction.
  • df: treatment \((t-1)\), block \((b-1)\), error \((t-1)(b-1)\).
  • CD = \(t_{\alpha/2, df_E}\sqrt{2\,\text{MS}_E/b}\).
  • Missing value: Yates' formula \(\hat y_{ij} = (tT'_i + bB'_j - G')/[(t-1)(b-1)]\); reduce dfE by 1.

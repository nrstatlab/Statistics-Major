Practical Course — Lab Manual

B.Sc. Statistics (Honours) — Semester V, Course 13 B Practical (1 Credit)

Laboratory Framework & Guidelines

Lab Requirements: This practical component demands 2 hours per week in the computer lab. Every student must personally perform the calculations, handle data inputs, interpret output results, and document their findings in a dedicated practical record book.

Prescribed List of Experiments

The following 12 experiments map directly to the core curriculum and form the evaluation framework for the practical semester examinations:

Phase 1: Empirical Fitting of Growth Curves

Experiment 1: Fitting of Modified Exponential Curve (Three Selected Points)

Objective: Fit the trend model \(Y_t = a + b c^t\) using three equidistant selected time markers.

Methodology: Select three points \(t_1, t_2, t_3\) separated by a fixed interval \(k\) such that \(t_2 - t_1 = t_3 - t_2 = k\). Utilize the explicit algebraic relations to extract parameters:

\[ c = \left(\dfrac{Y_3 - Y_2}{Y_2 - Y_1}\right)^{1/k}, \quad a = \dfrac{Y_1 Y_3 - Y_2^2}{Y_1 + Y_3 - 2 Y_2} \]

Experiment 2: Fitting of Modified Exponential Curve (Partial Sums Method)

Objective: Fit the trend model \(Y_t = a + b c^t\) over a series of \(n = 3k\) successive historical records.

Methodology: Partition the time-series dataset into three equal chronological groups of length \(k\). Compute the partial aggregates:

\[ S_1 = \sum_{t=1}^{k} Y_t, \quad S_2 = \sum_{t=k+1}^{2k} Y_t, \quad S_3 = \sum_{t=2k+1}^{3k} Y_t \]

Solve simultaneously for parameters using the smoothed group totals to minimize irregular variations.

Experiment 3: Fitting of Gompertz Curve (Three Selected Points)

Objective: Establish parameters for the asymmetric S-shaped growth curve \(Y_t = k a^{b^t}\) utilizing equispaced point markers.

Methodology: Apply a logarithmic transformation layer to linearize the model into a modified exponential form: \(\log Y_t = \log k + b^t \log a\). Solve for the ratio parameter \(b\) across equidistant steps using:

\[ b^k = \dfrac{\log Y_3 - \log Y_2}{\log Y_2 - \log Y_1} \]

Experiment 4: Fitting of Gompertz Curve (Partial Sums Method)

Objective: Fit the model \(Y_t = k a^{b^t}\) leveraging group totals across an entire time-series range.

Methodology: Convert all historical inputs into raw logarithmic entries (\(\log Y_t\)). Divide into three equal sequential blocks of length \(k\), find the block sums of logs, and calculate the upper saturation limit \(\log k\).

Experiment 5: Fitting of Logistic Curve (Partial Sums Method)

Objective: Fit the symmetric S-shaped growth framework \(Y_t = \dfrac{k}{1 + b e^{-ct}}\) to active population or technology distribution data arrays.

Methodology: Execute a reciprocal transformation layer \(Z_t = 1/Y_t\) to convert the logistic equation into a standard modified exponential curve format. Divide the converted series into three equal sub-groups to compute the carrying capacity parameter \(k\).

Phase 2: Advanced Index Number Operations

Experiment 6: Finding Index Numbers by Shifting the Base

Objective: Recalibrate a historical index number series onto a modern reference base year or a multi-year average base span.

Methodology: Standardize trackers by mapping each entry against the chosen reference value via the operational transformation:

\[ \text{New Index}_t = \dfrac{\text{Old Index}_t}{\text{Old Index in the New Base Year}} \times 100 \]

Experiment 7: Calculation of Fixed-Base and Chain-Base Index Numbers

Objective: Construct and contrast fixed-base series against dynamic chain-linked index structures.

Methodology: Determine year-on-year link relatives from underlying data points via \(L_t = \left(\dfrac{P_t}{P_{t-1}}\right) \times 100\). Link these relatives sequentially to build the continuous chain index series:

\[ \text{CBI}_t = \dfrac{\text{CBI}_{t-1} \times L_t}{100} \]

Experiment 8: Splicing Two Index Number Series

Objective: Join two non-overlapping or revised index number series sharing different baselines into a continuous historical timeline.

Methodology: Locate the common overlap row entry. Execute forward splicing to express older indices on the new base, or implement backward splicing to extend the current series into past decades:

\[ \text{Spliced New Index (Backward)} = \dfrac{\text{New Index}_t \times \text{Old Index in Overlap Year}}{100} \]

Experiment 9: Deflating Index Numbers

Objective: Adjust nominal monetary values (such as wages or national income data) to compute real purchasing power changes over time.

Methodology: Correct raw data series to eliminate inflationary distortion elements using the classical deflation formula:

\[ \text{Real Value}_t = \dfrac{\text{Nominal Value}_t}{\text{Price Index}_t} \times 100 \]

Phase 3: Demand Analysis and Educational Test Analytics

Experiment 10: Calculation of Income and Price Elasticities

Objective: Estimate demand responsiveness properties from household budget cross-sections or market pricing records.

Methodology: Determine partial price elasticities via point derivative models or arc methods:

\[ E_p = \dfrac{\Delta Q}{\Delta P} \cdot \dfrac{(P_1 + P_2)}{(Q_1 + Q_2)} \]

Fit log-linear regression layouts to extract constant demand parameters.

Experiment 11: Fitting of Pareto's Curve to Given Income Distributions

Objective: Fit Pareto’s distribution rule for higher wealth brackets to evaluate socioeconomic inequality.

Methodology: Map income values \(X\) and cumulative individuals exceeding that threshold \(N\) onto a double-log coordinate space. Fit a linear model via least-squares regression:

\[ \log N = \log A - \alpha \log X \]

Extract the slope parameter \(\alpha\) to evaluate concentration characteristics.

Experiment 12: Test Reliability Computations

Objective: Evaluate the internal consistency and measurement accuracy profiles of educational testing instruments.

Methodology: Utilizing item-difficulty values, sample means, and test score variance (\(\sigma_X^2\)), compute internal reliability coefficients using split-half layouts (with Spearman-Brown adjustments), the Rulon method, or the Kuder-Richardson formula (KR-20):

\[ r_{KR\text{-}20} = \dfrac{k}{k - 1}\left(1 - \dfrac{\sum_{i=1}^{k} p_i q_i}{\sigma_X^2}\right) \]

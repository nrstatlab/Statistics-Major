Experiment 1 — Moments of Uniform Distribution

PROBLEM

For \(X \sim U(2, 8)\), calculate first four central moments.

Solution: \(a = 2,\; b = 8;\; b - a = 6;\; \mu = (a+b)/2 = 5\).

\(\mu_2 = (b-a)^2/12 = 36/12 = 3\) (variance).

\(\mu_3 = 0\) (symmetric).

\(\mu_4 = (b-a)^4/80 = 1296/80 = 16.2\).

SD \(= \sqrt{3} = 1.732\); \(M\!D = (b-a)/4 = 1.5\).

Second problem

PROBLEM

If \(X \sim U(0, 10)\), find \(E(X^3)\).

\(E(X^3) = \int_0^{10} x^3/10\,dx = (10^4)/(40) = 250\).

Experiment 2 — Skewness & Kurtosis of Uniform

For any \(U(a, b)\):

\(\beta_1 = \mu_3^2/\mu_2^3 = 0\) — perfectly symmetric.

\(\beta_2 = \mu_4 / \mu_2^2 = 16.2/9 = 1.8\) — strongly platykurtic. \(\gamma_2 = -1.2\).

The result is independent of \(a, b\) — uniform always has the same shape.

Experiment 4 — Gamma Distribution Applications

PROBLEM 1

Let \(X \sim G(3, 2)\) (waiting time for the third call when calls arrive at rate 2/hour). Find \(P(X \le 1)\) and \(E(X)\).

\(E(X) = 3/2 = 1.5\) hours; \(\text{Var}(X) = 3/4 = 0.75\).

\(f(x) = 4 x^2 e^{-2x};\quad P(X \le 1) = \int_0^1 4 x^2 e^{-2x}\,dx\).

Using integration by parts (or recurrence): \(P(X \le 1) = 1 - e^{-2}(1 + 2 + 2) = 1 - 5e^{-2} \approx 0.3233\).

PROBLEM 2

If \(X_1 \sim G(2, 1)\) and \(X_2 \sim G(3, 1)\) are independent, find the distribution of \(X_1 + X_2\).

By the additive property (same \(\lambda\)): \(X_1 + X_2 \sim G(5, 1)\). Mean = 5; Variance = 5.

Experiment 6 — Fitting Normal (Ordinates Method)

Use the same \(\hat \mu, \hat \sigma\) as in Experiment 5. Instead of computing class probabilities, evaluate the ordinate \(\phi((x_i - \hat\mu)/\hat\sigma)\) at the mid-point of each class and approximate

\[ \hat f_i \;\approx\; \dfrac{N \cdot h}{\hat \sigma} \cdot \phi\!\left(\dfrac{x_i - \hat \mu}{\hat \sigma}\right), \]

where \(h\) is class width and \(\phi(z) = \dfrac{1}{\sqrt{2\pi}}e^{-z^2/2}\) is the standard normal PDF.

For our data \(N = 200, h = 5, \hat\sigma = 6.2\); coefficient \(N h/\hat\sigma = 161.29\).

Experiment 7 — Standard Normal Problems

PROBLEM 1

Marks of 1000 students follow \(N(60, 100)\). How many students score between 50 and 75?

\(z_1 = (50-60)/10 = -1;\; z_2 = (75-60)/10 = 1.5\).

\(P = \Phi(1.5) - \Phi(-1) = 0.9332 - 0.1587 = 0.7745\). Number = 0.7745 × 1000 = 774.

PROBLEM 2

The IQ of a population follows \(N(100, 225)\). Find the IQ value that is exceeded by only 5 % of the population.

Need \(P(X > x) = 0.05\), i.e., \(\Phi(z) = 0.95\) ⇒ \(z = 1.645\).

\(x = \mu + z\sigma = 100 + 1.645 \times 15 = 124.7\). So an IQ of about 125 marks the top 5 %.

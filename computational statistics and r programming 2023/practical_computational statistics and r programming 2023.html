Practical Course — 11 Exercises

Hands-on R exercises with complete, ready-to-run scripts.

How to use: open RStudio, create a new script, type each block, and run it line-by-line with Ctrl + Enter. Predict each result before you run it.

Exercise 1: Installing R and RStudio

STEPS
  1. Download and install R from the CRAN website (cran.r-project.org).
  2. Download and install RStudio Desktop (free).
  3. Open RStudio; in the Console type R.version.string to confirm the version.
R.version.string     # e.g. "R version 4.4.1 ..."
sessionInfo()        # full environment details

Exercise 6: Working with Vectors

SCRIPT 
v1 <- 1:20            # (a) elements 1 to 20
v1 + 2               # (b) add 2 to every element
v1 / 5               # (c) divide every element by 5
v2 <- 21:30          # (d) elements 21 to 30
v1 + v2              # add v1 to v2 (recycling: v2 length 10 recycled over v1 length 20)

Note: v1 has 20 elements and v2 has 10, so R recycles v2 twice when adding — a deliberate illustration of the recycling rule.

Exercise 7: Distance Matrix — Shortest Pair

SCRIPT 
M <- matrix(c(0,12,13,8,20,
              12,0,15,28,88,
              13,15,0,6,9,
              8,28,6,0,33,
              20,88,9,33,0),
            nrow = 5, byrow = TRUE)
rownames(M) <- colnames(M) <- paste0("C", 1:5)
M
min_dist <- min(M[M > 0])               # smallest non-zero distance = 6
which(M == min_dist, arr.ind = TRUE)    # rows/cols -> C3 & C4

Result: the shortest distance is 6, between cities C3 and C4.

Exercise 8: Student Marks Data Frame

SCRIPT 
marks <- data.frame(
  section = c("A","A","A","B","B","B"),
  student = c(1,2,3,1,2,3),
  M1 = c(46,34,56,43,67,76),
  M2 = c(54,55,66,44,76,68),
  M3 = c(45,55,64,45,78,37)
)
# (a) stored with proper names above
# (b) marks and totals for all students
marks$total <- marks$M1 + marks$M2 + marks$M3
print(marks)
# (c) highest total in each section
aggregate(total ~ section, data = marks, FUN = max)
# (d) add a new subject M4 with marks for both sections
marks$M4 <- c(50, 60, 70, 55, 65, 40)
marks$total <- marks$M1 + marks$M2 + marks$M3 + marks$M4
print(marks)

Exercise 9: Shop-Cost Matrices — Cheapest Shop

SCRIPT 
# (a) price matrix: rows = items, cols = shops S1, S2
price <- matrix(c(1.5, 1,
                  2,   2.5,
                  5,   4.5,
                  16,  17),
                nrow = 4, byrow = TRUE,
                dimnames = list(c("Roll","Bun","Cake","Bread"), c("S1","S2")))
# demand matrix: rows = persons, cols = items
demand <- matrix(c(6,5,3,1,
                   3,6,3,2,
                   3,4,3,1),
                 nrow = 3, byrow = TRUE,
                 dimnames = list(c("P1","P2","P3"),
                                 c("Roll","Bun","Cake","Bread")))
# (b) display
price; demand
# (c) total cost for each person in each shop
cost <- demand %*% price          # 3x2 matrix
cost
# (d) suggest the cheaper shop for each person
apply(cost, 1, function(r) names(which.min(r)))

Matrix multiplication demand %*% price gives each person's total spend in each shop; which.min picks the cheaper one per person.

Exercise 10: summary() for Descriptive Statistics

SCRIPT 
x <- c(46, 54, 45, 34, 55, 64, 78, 90, 41, 66)
summary(x)        # Min, 1st Qu, Median, Mean, 3rd Qu, Max
mean(x); median(x); sd(x); var(x)
range(x); IQR(x)

summary() returns the five-number summary plus the mean in one call; the other functions give the individual statistics (mean, median, SD, variance, range, inter-quartile range).

Exercise 11: Visualizations

SCRIPT 
vals <- c(5, 8, 12, 6, 9)
labels <- c("A","B","C","D","E")

# Bar chart
barplot(vals, names.arg = labels, col = "steelblue", main = "Bar Chart")

# Histogram
hist(rnorm(200, mean = 50, sd = 10), col = "lightblue", main = "Histogram")

# Box plot
boxplot(vals, main = "Box Plot", col = "lightgreen")

# Line chart
plot(1:5, vals, type = "o", col = "darkred", main = "Line Chart", xlab = "Index", ylab = "Value")

# Scatter plot
study <- c(2,4,6,8,10);  score <- c(15,23,28,36,43)
plot(study, score, pch = 19, main = "Scatter Plot")
abline(lm(score ~ study), col = "blue")   # add regression line

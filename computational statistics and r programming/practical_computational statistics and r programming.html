Practical Course — 7 Experiments

Semester V, Course 11 (Practical) — 2 hrs/week, Credit 1. Hands-on R labs to be done in the Computer Lab (≥ 4 hrs/month).

List of Practical Experiments (Official Syllabus)

  1. Data Import and Preprocessing — import dataset (e.g., iris), handle missing values, basic sub-setting.
  2. Descriptive Statistics Computation — mean, variance, skewness; interpret results.
  3. Frequency Analysis — contingency tables and chi-square tests on categorical data.
  4. Basic Visualizations — histograms and boxplots to assess data normality.
  5. Hypothesis Testing — t-tests and ANOVA on experimental data, reporting p-values.
  6. Non-Parametric Tests — Wilcoxon tests; compare with parametric alternatives.
  7. Linear Regression Modeling — fit a simple linear model, plot residuals, predict new values.
Use built-in R datasets (iris, mtcars, airquality, ToothGrowth) or download CSV from the UCI ML Repository for these experiments.

Experiment 1 — Data Import & Preprocessing

AIM

Import a dataset, examine its structure, handle missing values, perform sub-setting.

# Built-in dataset
data(airquality)
str(airquality)              # 153 obs of 6 vars; NAs in Ozone and Solar.R

# Or read a CSV (uncomment and edit path)
# air <- read.csv("airquality.csv")

# Check missing values
colSums(is.na(airquality))   # Ozone:37, Solar.R:7

# Drop rows with any NA
clean <- na.omit(airquality)
nrow(clean)                   # 111

# Replace Ozone NAs with mean
airquality$Ozone[is.na(airquality$Ozone)] <- mean(airquality$Ozone, na.rm = TRUE)

# Subset: rows where temperature > 80
hot <- subset(airquality, Temp > 80)
nrow(hot)

# Save preprocessed data
write.csv(clean, "airquality_clean.csv", row.names = FALSE)

Experiment 3 — Frequency Analysis & Chi-Square

3a — One-way frequency table

# Categorise iris by petal length (long if > 4)
big <- ifelse(iris$Petal.Length > 4, "Long", "Short")
table(iris$Species, big)
big Long Short setosa 0 50 versicolor 34 16 virginica 50 0

3b — Chi-Square Test of Independence

tbl <- table(iris$Species, big)
chisq.test(tbl)
#  X-squared = 105.84, df = 2, p-value < 2.2e-16
# Strong association ⇒ reject H0 of independence

3c — Goodness-of-fit (fair die simulation)

set.seed(7); rolls <- sample(1:6, 120, replace = TRUE)
observed <- table(rolls); observed
chisq.test(observed)           # expected: uniform 1/6 each

#  p-value = 0.0102 ⇒ borderline; histogram looks roughly normal but tails differ

Experiment 5 — Hypothesis Testing (t-tests & ANOVA)

One-sample t-test

x <- c(48, 52, 49, 53, 51, 47, 55, 50, 49, 52)
t.test(x, mu = 50)              # p = 0.4601 ⇒ fail to reject H0

Two-sample t-test (Welch)

setosa     <- iris$Sepal.Length[iris$Species == "setosa"]
versicolor <- iris$Sepal.Length[iris$Species == "versicolor"]
t.test(setosa, versicolor)        # p < 2.2e-16 ⇒ reject H0

Paired t-test (sleep dataset)

data(sleep)
t.test(extra ~ group, data = sleep, paired = TRUE)
#  t = -4.062, df = 9, p-value = 0.00283 ⇒ Drug effect significant

One-way ANOVA

fit <- aov(Sepal.Length ~ Species, data = iris)
summary(fit)
TukeyHSD(fit)                     # all 3 pairs significant

Two-way ANOVA

data(ToothGrowth)
fit2 <- aov(len ~ supp * factor(dose), data = ToothGrowth)
summary(fit2)

Experiment 6 — Non-Parametric Tests

Wilcoxon Signed-Rank (one-sample / paired)

# One-sample
x <- c(48, 52, 49, 53, 51, 47, 55, 50, 49, 52)
wilcox.test(x, mu = 50)                    # V = 28.5, p = 0.511
# (R warns about ties — with tied ranks it uses a normal approximation)

# Paired sleep data
wilcox.test(extra ~ group, data = sleep, paired = TRUE)

Wilcoxon Rank-Sum (two-sample)

wilcox.test(setosa, versicolor)            # W = 168.5, p = 8.3e-14 ⇒ groups differ

Compare with parametric alternative

t_result <- t.test(setosa, versicolor)
w_result <- wilcox.test(setosa, versicolor)

data.frame(
  Test  = c("Welch t-test","Wilcoxon"),
  p_val = c(t_result$p.value, w_result$p.value)
)
# Both reject H0; Wilcoxon is appropriate when normality is doubtful.

Experiment 7 — Linear Regression Modeling

# Simple linear regression: MPG vs Weight
data(mtcars)
fit <- lm(mpg ~ wt, data = mtcars)
summary(fit)
Coefficients: Estimate Std. Error t value Pr(>|t|) (Intercept) 37.285 1.878 19.86 <2e-16 *** wt -5.344 0.559 -9.56 1.29e-10 *** Multiple R-squared: 0.753

Scatter with fitted line

plot(mtcars$wt, mtcars$mpg, pch = 19, col = "navy",
     main = "MPG vs Weight",
     xlab = "Weight (1000 lbs)", ylab = "Miles per Gallon")
abline(fit, col = "red", lwd = 2)

Residual diagnostics

par(mfrow = c(2, 2))
plot(fit)
par(mfrow = c(1, 1))

shapiro.test(resid(fit))   # check residual normality
hist(resid(fit), breaks = 12, col = "lightgray",
     main = "Distribution of Residuals", xlab = "Residual")

Predict new values

new <- data.frame(wt = c(2.5, 3.5, 4.5))
predict(fit, new, interval = "prediction")
fit lwr upr 1 23.92 17.55 30.29 2 18.58 12.25 24.90 3 13.24 6.75 19.72

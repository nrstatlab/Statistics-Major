Practical Course — 8 Experiments

Semester V, Course 13 A (Practical) — 2 hrs/week, Credit 1. Worked problems on TP, AP, sequencing, CPM/PERT and game theory.

List of Practical Experiments (Official Syllabus)

  1. IBFS of transportation problem using North-West Corner rule, LCEM and VAM.
  2. Optimum solution to balanced & unbalanced transportation problems by MODI method (minimization cases).
  3. Solution of assignment problem using Hungarian method (minimization cases).
  4. Solution of sequencing problem — n jobs through two machines.
  5. Solution of sequencing problem — n jobs through three machines.
  6. Project scheduling of a given project (deterministic case — CPM).
  7. Project scheduling of a given project (probabilistic case — PERT).
  8. Solution of m × n games by dominance rule.

Experiment 1 — IBFS by NWCM, LCEM and VAM

PROBLEM

Transportation problem (supply, demand, costs as below):

D₁D₂D₃D₄Supply
S₁3174300
S₂2659400

Demand2503504002001200

NWCM Solution

Allocations: x₁₁=250, x₁₂=50, x₂₂=300, x₂₃=100, x₃₃=300, x₃₄=200. Cost = 250·3 + 50·1 + 300·6 + 100·5 + 300·3 + 200·2 = 4400.

LCEM Solution

Allocations: x₁₂=300, x₂₁=250, x₃₄=200, x₃₂=50, x₃₃=250, x₂₃=150. Cost = 300·1 + 250·2 + 200·2 + 50·3 + 250·3 + 150·5 = 2850.

VAM Solution

Initial penalties (row): 2, 3, 1; (col): 1, 2, 2, 2. Largest = 3 (row S₂). Min in S₂ = 2 at D₁, allocate 250. Continue with recomputed penalties.

Final VAM allocations typically yield cost ≈ 2700 — often the optimum or very close.

Experiment 2 — MODI Method (Balanced & Unbalanced)

Balanced Case

Starting from VAM solution above. Compute u_i, v_j from basic cells using \(u_i + v_j = c_{ij}\). Then \(d_{ij} = c_{ij} - (u_i + v_j)\) for non-basic cells. If any \(d_{ij} < 0\), enter a closed loop and adjust by θ.

Continue until all \(d_{ij} \ge 0\). Final optimum cost: 2670 (after MODI improvement).

Unbalanced Case

Supply: 100, 150; Demand: 80, 100, 90. Total supply = 250, demand = 270 ⇒ add dummy source with supply 20, cost 0.

After balancing, apply VAM then MODI as in the balanced case.

Same network with three time estimates per activity:

Critical path A → C → F: T_e = 3 + 7 + 4 = 14 days; σ² = 0.111 + 0.444 + 0.111 = 0.666; σ = 0.816.

P(finish in 15 days) = Φ((15 − 14)/0.816) = Φ(1.225) = 0.890.

P(finish in 16 days) = Φ((16 − 14)/0.816) = Φ(2.449) = 0.993.

Time to achieve 95% probability: T = 14 + 1.645 × 0.816 = 15.34 days.

Experiment 8 — Game Theory by Dominance Rule

PROBLEM

Solve the m × n game:

B₁B₂B₃B₄
A₁3240
A₂3424
A₃4240
A₄0408

Step 1 — Check for saddle point

Row mins: 0, 2, 0, 0. Col maxes: 4, 4, 4, 8. Maxmin = 2, Minimax = 4 ⇒ no saddle.

Step 2 — Apply Dominance

Compare A₁ and A₃: (3, 2, 4, 0) vs (4, 2, 4, 0). A₃ ≥ A₁ everywhere ⇒ A₁ is dominated by A₃. Delete A₁.

Reduced:

B₁B₂B₃B₄
A₂3424
A₃4240
A₄0408

Compare B₁ and B₃: (3, 4, 0) vs (2, 4, 0). For B (minimiser), smaller is better. B₃ ≤ B₁ everywhere ⇒ B₁ is dominated. Delete B₁.

Continue: compare B₂ vs B₄: (4, 2, 4) vs (4, 0, 8). B₂ not dominated. Try mixed dominance — average of B₂ and B₃ = (3, 3, 2). Compare with B₄ (4, 0, 8): no clear dominance.

Step 3 — Apply 2×2 sub-game or graphical method

Reduced matrix may still be 3×3. Continue with mixed dominance or LP. After applying all dominance, the game often reduces to 2×2 — solve with closed-form.

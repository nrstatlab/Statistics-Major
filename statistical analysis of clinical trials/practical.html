Experiment 1: Determination of Sample Size

OBJECTIVE

To determine the sample size required for a clinical trial comparing two independent groups with (a) a dichotomous response variable and (b) a continuous response variable, given specified significance level, power, and effect size.

Part A: Dichotomous Response Variable

Scenario: A Phase III trial compares a new drug with a placebo for the prevention of post-operative nausea and vomiting (PONV). The current standard has a response rate (no PONV) of approximately \(p_C = 0.45\). The investigators consider the new drug clinically meaningful if it increases the response rate to \(p_T = 0.65\). Design the trial at \(\alpha = 0.05\) (two-sided) and 80% power.

SAMPLE SIZE CALCULATION

\[ \bar{p} = \frac{p_T + p_C}{2} = \frac{0.65 + 0.45}{2} = 0.55, \quad \bar{q} = 1 - 0.55 = 0.45 \]

\[ n = \frac{(z_{\alpha/2}\sqrt{2\bar{p}\bar{q}} + z_\beta\sqrt{p_T q_T + p_C q_C})^2}{(p_T - p_C)^2} \]

\[ = \frac{(1.96\sqrt{2 \times 0.55 \times 0.45} + 0.842\sqrt{0.65 \times 0.35 + 0.45 \times 0.55})^2}{(0.20)^2} \]

\[ = \frac{(1.96 \times 0.7036 + 0.842 \times 0.6892)^2}{0.04} = \frac{(1.379 + 0.580)^2}{0.04} = \frac{3.838}{0.04} = 95.9 \]

Rounding up: 96 subjects per group (192 total).

Adjusting for 10% dropout: \(n_{\text{adj}} = 96/0.90 = 107\) per group (214 total).

Part B: Continuous Response Variable

Scenario: A trial compares a new antihypertensive with a placebo. The primary endpoint is the change in systolic blood pressure (SBP) from baseline to week 8. From previous studies, \(\sigma = 12\) mmHg. The minimum clinically meaningful difference is \(\delta = 5\) mmHg. Design at \(\alpha = 0.05\) (two-sided) and 90% power.

SAMPLE SIZE CALCULATION

\[ n = \frac{2\sigma^2(z_{\alpha/2} + z_\beta)^2}{\delta^2} = \frac{2 \times 144 \times (1.96 + 1.282)^2}{25} \]

\[ = \frac{288 \times 10.507}{25} = \frac{3026.0}{25} = 121.0 \]

Rounding up: 122 subjects per group (244 total).

With 10% dropout: \(122/0.90 = 136\) per group (272 total).

R Code

R IMPLEMENTATION 
# Part A: Dichotomous response
p_T <- 0.65; p_C <- 0.45
alpha <- 0.05; power <- 0.80
n_dich <- power.prop.test(p1 = p_T, p2 = p_C,
           sig.level = alpha, power = power)$n
cat("Dichotomous: n per group =", ceiling(n_dich), "\n")

# Part B: Continuous response
delta <- 5; sigma <- 12
n_cont <- power.t.test(delta = delta, sd = sigma,
           sig.level = alpha, power = 0.90)$n
cat("Continuous: n per group =", ceiling(n_cont), "\n")

Experiment 4: Analysis of Clinical Trial Data using Parallel Design

OBJECTIVE

To analyse data from a parallel-group clinical trial, perform the two-sample t-test for continuous outcomes and the chi-square test for categorical outcomes, construct confidence intervals, and interpret the results in a clinical context.

Scenario: A Phase III parallel-group trial compares a new antidiabetic drug (Drug A) with the standard drug (Drug B) in 100 patients with type 2 diabetes. The primary endpoint is the change in HbA1c (%) from baseline to 12 weeks.

Data summary:

Drug A (n=50)Drug B (n=50)
Mean ΔHbA1c (%)−1.82−1.24
SD0.951.02
PARALLEL DESIGN ANALYSIS

Two-sample t-test:

\[ \text{Pooled variance: } s_p^2 = \frac{49 \times 0.9025 + 49 \times 1.0404}{98} = \frac{44.22 + 50.98}{98} = 0.9714 \]

\[ s_p = 0.9856 \]

\[ t = \frac{-1.82 - (-1.24)}{0.9856 \times \sqrt{\frac{1}{50} + \frac{1}{50}}} = \frac{-0.58}{0.9856 \times 0.2000} = \frac{-0.58}{0.1971} = -2.943 \]

With \(df = 98\), two-sided p-value = 0.004.

95% CI for the difference:

\[ -0.58 \pm 1.984 \times 0.1971 = -0.58 \pm 0.391 = (-0.971, -0.189) \]

Effect size (Cohen's d):

\[ d = \frac{-0.58}{0.9856} = -0.588 \]

A medium-to-large effect size. Drug A produces a statistically significantly greater reduction in HbA1c compared to Drug B, with the 95% CI indicating the true difference is between 0.19 and 0.97 percentage points.

Categorical analysis (HbA1c < 7% at week 12):

ControlledNot ControlledTotal
Drug A321850
Drug B212950

\[ \chi^2 = \frac{100 \times (32 \times 29 - 18 \times 21)^2}{50 \times 50 \times 53 \times 47} = \frac{100 \times (928 - 378)^2}{6227500} = \frac{100 \times 302500}{6227500} = 4.86 \]

\(df = 1\), p = 0.028. Drug A has a significantly higher proportion of patients achieving glycaemic control (64%) vs. Drug B (42%).

\[ \text{OR} = \frac{32 \times 29}{18 \times 21} = \frac{928}{378} = 2.45 \]

Patients on Drug A have 2.45 times the odds of achieving glycaemic control.

Experiment 5: Meta-Analysis of Clinical Trials

OBJECTIVE

To perform a meta-analysis of multiple clinical trials, compute pooled effect estimates using both fixed-effect and random-effects models, assess heterogeneity, construct a forest plot, and evaluate potential publication bias.

Scenario: A meta-analysis examines the effect of aspirin vs. placebo on the prevention of cardiovascular events (myocardial infarction, stroke, or cardiovascular death) in high-risk patients. Six randomised controlled trials are identified.

Study data:

META-ANALYSIS CALCULATIONS

Step 1: Compute OR and ln(OR) for each trial:

For trial1: \(a = 148\), \(b = 3023\), \(c = 186\), \(d = 2982\).

\[ \text{OR}_1 = \frac{148 \times 2982}{3023 \times 186} = \frac{441336}{562278} = 0.785 \]

\[ \ln(\text{OR}_1) = -0.242, \quad SE_1 = \sqrt{\frac{1}{148} + \frac{1}{3023} + \frac{1}{186} + \frac{1}{2982}} = 0.112 \]

Step 2: Fixed-effect pooled estimate:

\[ \hat{\theta}_{FE} = -0.199 \quad (\text{pooled ln(OR)}) \]

Pooled OR = \(\exp(-0.199) = 0.820\) (95% CI: 0.754–0.891), p < 0.001.

Step 3: Heterogeneity:

\[ Q = 3.42, \quad df = 5, \quad p = 0.636 \]

\[ I^2 = 0\% \quad (95\% \text{ CI: } 0\%, 56\%) \]

No significant heterogeneity — all trials show a consistent treatment effect.

Step 4: Random-effects model:

Since \(Q < k - 1\), \(\hat{\tau}^2 = 0\), and the random-effects model gives identical results to the fixed-effect model: pooled OR = 0.820 (95% CI: 0.754–0.891).

Step 5: Publication bias:

Egger's test: intercept = −0.42 (SE = 0.85), t = −0.49, p = 0.65. No evidence of publication bias. The funnel plot is approximately symmetric.

R Code

R IMPLEMENTATION 
library(meta)

# Enter data
trials <- c("trial1", "trial2", "trial3", "trial4", "trial5", "trial6")
n_trt <- c(3171, 2035, 2530, 3402, 5139, 4401)
e_trt <- c(148, 109, 183, 209, 298, 265)
n_ctrl <- c(3168, 2037, 2540, 3395, 5127, 4405)
e_ctrl <- c(186, 142, 212, 251, 347, 302)

# Meta-analysis (binary outcomes)
meta_result <- metabin(event.e = e_trt, n.e = n_trt,
                        event.c = e_ctrl, n.c = n_ctrl,
                        studlab = trials,
                        sm = "OR",
                        method = "MH",
                        method.tau = "DL")

# Print results
print(meta_result)

# Forest plot
forest(meta_result)

# Heterogeneity
cat("I² =", meta_result$I2, "%\n")
cat("Q =", meta_result$Q, ", p =", meta_result$p.val.Q, "\n")

# Publication bias: funnel plot and Egger's test
funnel(meta_result)
metabias(meta_result, method = "egger")

# Trim-and-fill
tf <- trimfill(meta_result)
print(tf)
funnel(tf)

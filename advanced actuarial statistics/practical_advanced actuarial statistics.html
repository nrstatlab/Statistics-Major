Practical Course — 10 Experiments

Hands-on actuarial computations using life-table data, force-of-mortality functions and recursion.

Notation Used Throughout

SymbolMeaning
\(l_x\)Number of lives age \(x\) per starting cohort (radix usually 10,000 or 100,000).
\(q_x\)\((l_x - l_{x+1})/l_x\); probability of dying in year of age \(x\).
\(p_x\)\(1 - q_x\); probability of surviving from \(x\) to \(x+1\).
\(i, v, \delta\)Effective annual rate, discount factor \(v = 1/(1+i)\), force of interest \(\delta = \ln(1+i)\).

Experiment 1: Fitting Gompertz's Law

OBJECTIVE

Estimate the Gompertz parameters \(B, c\) from a set of empirical \(\mu_x\) values.

Data & Procedure

  1. Take \(\mu_x\) at three or four ages (say \(x = 40, 50, 60, 70\)).
  2. Linearise: \(\ln\mu_x = \ln B + x \ln c\). Plot \(\ln\mu_x\) versus \(x\).
  3. Slope of the line = \(\ln c\); intercept = \(\ln B\).
WORKED EXAMPLE

Data: \(\mu_{40} = 0.003\), \(\mu_{50} = 0.008\), \(\mu_{60} = 0.020\), \(\mu_{70} = 0.052\).

\(\ln\mu_x\): -5.809, -4.828, -3.912, -2.957. Differences ≈ +0.95 per 10-year jump, so \(\ln c \approx 0.095\), giving \(c = e^{0.095} = 1.0997 \approx 1.10\).

\(\ln B = \ln\mu_{40} - 40\ln c = -5.809 - 40 \times 0.095 = -9.609\), so \(B = e^{-9.609} = 0.0000671\).

Gompertz fit: \(\mu_x \approx 0.0000671 \times 1.10^x\).

Experiment 2: Construction of a Life Table from \(q_x\)

  1. Start with a radix, e.g. \(l_0 = 10{,}000\).
  2. Apply \(l_{x+1} = l_x \cdot (1 - q_x)\) recursively.
  3. Tabulate \(l_x\) and \(d_x = l_x - l_{x+1}\) for the ages required.
WORKED EXAMPLE

\(q_{30} = 0.002, q_{31} = 0.0022, q_{32} = 0.0024\); \(l_{30} = 9700\).

\(l_{31} = 9700 \times 0.998 = 9680.6\); \(l_{32} = 9680.6 \times 0.9978 = 9659.3\); \(l_{33} = 9659.3 \times 0.9976 = 9636.1\).

Deaths: \(d_{30} = 19.4, d_{31} = 21.3, d_{32} = 23.2\).

Experiment 3: Computation of \(A_x\) by Recursion

Use \(A_x = v\,q_x + v\,p_x\,A_{x+1}\), starting from an assumed limiting age \(\omega\) with \(A_\omega = v\), and recurse downwards.

WORKED EXAMPLE

\(\omega = 100\), \(i = 0.05\), so \(v = 0.9524\). Take \(q_{97} = 0.40, q_{98} = 0.50, q_{99} = 0.60\).

\(A_{100} = 0.9524.\)

\(A_{99} = 0.9524 \times 0.60 + 0.9524 \times 0.40 \times 0.9524 = 0.5714 + 0.3629 = 0.9343.\)

\(A_{98} = 0.9524 \times 0.50 + 0.9524 \times 0.50 \times 0.9343 = 0.4762 + 0.4450 = 0.9212.\)

\(A_{97} = 0.9524 \times 0.40 + 0.9524 \times 0.60 \times 0.9212 = 0.3810 + 0.5265 = 0.9075.\)

Experiment 4: \(\bar A_x\) via the UDD Factor

If \(A_x\) and \(i\) are known, compute \(\bar A_x = (i/\delta)\,A_x\).

WORKED EXAMPLE

\(A_{55} = 0.2750\), \(i = 0.06\); \(\delta = \ln 1.06 = 0.05827\); \(i/\delta = 1.02971\).

\(\bar A_{55} = 1.02971 \times 0.2750 = 0.28317.\)

Experiment 5: Computation of \(\ddot a_x\)

Use the identity \(\ddot a_x = (1 - A_x)/d\), where \(d = 1 - v\).

WORKED EXAMPLE

\(A_{55} = 0.2750, i = 0.06\); \(d = 0.06/1.06 = 0.05660\).

\(\ddot a_{55} = (1 - 0.2750)/0.05660 = 0.7250/0.05660 = 12.81.\)

\(a_{55} = \ddot a_{55} - 1 = 11.81\); \(\bar a_{55} \approx \ddot a_{55} - 0.5 = 12.31\).

Experiment 6: Net Annual Premium

WORKED EXAMPLE

Whole-life on (55), discrete contract, sum assured ₹10 lakh. From Experiment 5: \(A_{55} = 0.2750\), \(\ddot a_{55} = 12.81\).

\(P_{55} = 0.2750/12.81 = 0.02147\) per ₹1.

Annual premium = ₹10,00,000 × 0.02147 = ₹21,470.

Experiment 7: Reserve \({}_tV_x\) Computation

Use \({}_tV_x = A_{x+t} - P_x \cdot \ddot a_{x+t}\).

WORKED EXAMPLE

From Experiment 6: \(P_{55} = 0.02147\). At duration 5: \(A_{60} = 0.3300, \ddot a_{60} = 11.83\).

\({}_5V_{55} = 0.3300 - 0.02147 \times 11.83 = 0.3300 - 0.2540 = 0.0760.\)

For a ₹10 lakh policy the duration-5 reserve is ₹76,000.

Experiment 8: Thiele's Recursion (Reserve build-up)

Verify \({}_{t+1}V = [({}_tV + P)(1+i) - q_{x+t}\,b]/p_{x+t}\).

WORKED EXAMPLE

Whole-life on (55): \(P_{55} = 0.02147\), \(i = 0.06\), \(q_{55} = 0.008\), \(b = 1\), \({}_0V = 0\).

\((0 + 0.02147)(1.06) = 0.02276.\)

\(q_{55}\,b = 0.008.\)

\({}_1V = (0.02276 - 0.008)/0.992 = 0.01476/0.992 = 0.01488.\)

Iterate to build the entire reserve schedule year by year.

Experiment 9: Joint-Life and Last-Survivor APVs

WORKED EXAMPLE

Couple (65, 60). Independent lives. \(A_{65} = 0.40, A_{60} = 0.28, A_{65:60} = 0.46\). Compute:

\(A_{\overline{65:60}} = A_{65} + A_{60} - A_{65:60} = 0.40 + 0.28 - 0.46 = 0.22\).

Reversionary annuity (pays widow only after husband dies): given \(\ddot a_{65:60} = 10.9\), \(\ddot a_{60} = 14.8\), \(\ddot a_{65|60} = 14.8 - 10.9 = 3.9\).

Experiment 10: Gross-Premium Reserve with Expenses

WORKED EXAMPLE

20-year endowment on (40), sum assured 1. Expense assumption: initial expense \(E_0 = 0.04\) (4% of SA), renewal expense \(E_m = 0.005\) per year, premium-related expense \(e_p = 5\%\) of each premium.

From Unit 4: \(A_{40:\overline{20}|} = 0.336\), \(\ddot a_{40:\overline{20}|} = 11.45\), net premium \(P = 0.02934\).

Gross premium \(G\):

\(G(1 - 0.05) \times 11.45 = 0.336 + 0.04 + 0.005 \times 11.45\)

\(G \times 10.8775 = 0.336 + 0.04 + 0.05725 = 0.43325\)

\(G = 0.43325/10.8775 = 0.03984.\)

So the gross premium (0.03984) is 36% higher than the net premium (0.02934) — the loading covers the initial 4% expense, ongoing 0.5% maintenance, and 5%-of-premium commission.

Practical Course — Lab Manual

B.Sc. Statistics (Honours) — Semester VI, Course 14 A Practical (1 Credit)

Laboratory Framework & Guidelines

Lab Requirements: This practical component demands 2 hours per week in the computer lab. Every student must personally perform the calculations, construct control charts, compute acceptance sampling metrics, and document their findings in a dedicated practical record book.

Prescribed List of Experiments

The following 9 experiments map directly to the core curriculum and form the evaluation framework for the practical semester examinations:

Phase 1: Variable Control Charts

Experiment 1: Construction of Mean and R Charts

Objective: Construct the \(\bar{X}\) and \(R\) control charts for a variable-quality characteristic when standards are unspecified.

Methodology: Collect 20–25 subgroups of size \(n = 4\) or 5. Compute \(\bar{X}_j\) and \(R_j\) for each subgroup. Calculate \(\bar{\bar{X}}\) and \(\bar{R}\). Use constants \(A_2\), \(D_3\), \(D_4\) to set control limits. Plot both charts and interpret.

\[ \text{UCL}_{\bar{X}} = \bar{\bar{X}} + A_2 \bar{R}, \quad \text{LCL}_{\bar{X}} = \bar{\bar{X}} - A_2 \bar{R}; \qquad \text{UCL}_R = D_4 \bar{R}, \quad \text{LCL}_R = D_3 \bar{R}. \]

Experiment 2: Construction of Mean and Standard Deviation Charts

Objective: Construct the \(\bar{X}\) and \(S\) control charts when standards are unspecified.

Methodology: Compute \(S_j\) for each subgroup. Calculate \(\bar{S}\). Use constants \(A_3\), \(B_3\), \(B_4\) to establish control limits. Compare with the \(\bar{X}\)-R chart results from Experiment 1.

\[ \text{UCL}_{\bar{X}} = \bar{\bar{X}} + A_3 \bar{S}, \quad \text{LCL}_{\bar{X}} = \bar{\bar{X}} - A_3 \bar{S}; \qquad \text{UCL}_S = B_4 \bar{S}, \quad \text{LCL}_S = B_3 \bar{S}. \]

Phase 2: Attribute Control Charts

Experiment 3: Construction of p Chart for Fixed Sample Size

Objective: Monitor the fraction defective when all subgroups have the same size \(n\).

Methodology: Count defectives \(d_j\) in each subgroup. Compute \(\bar{p} = \sum d_j / (kn)\). Set 3-sigma limits:

\[ \text{UCL}_p = \bar{p} + 3\sqrt{\frac{\bar{p}(1-\bar{p})}{n}}, \qquad \text{LCL}_p = \bar{p} - 3\sqrt{\frac{\bar{p}(1-\bar{p})}{n}}. \]

Experiment 4: Construction of p Chart for Variable Sample Size

Objective: Monitor the fraction defective when subgroup sizes differ.

Methodology: Compute separate limits for each subgroup using its own \(n_j\). Alternatively, use the average sample size \(\bar{n}\) for approximate limits and verify borderline points with exact limits.

Experiment 5: Construction of np Chart

Objective: Monitor the number of defectives when subgroup size \(n\) is constant.

Methodology: Plot \(d_j\) directly. Compute limits:

\[ \text{UCL}_{np} = n\bar{p} + 3\sqrt{n\bar{p}(1-\bar{p})}, \qquad \text{LCL}_{np} = n\bar{p} - 3\sqrt{n\bar{p}(1-\bar{p})}. \]

Experiment 6: Construction of c Chart

Objective: Monitor the number of defects per constant inspection unit.

Methodology: Compute \(\bar{c}\) and set limits:

\[ \text{UCL}_c = \bar{c} + 3\sqrt{\bar{c}}, \qquad \text{LCL}_c = \bar{c} - 3\sqrt{\bar{c}}. \]

Experiment 7: Construction of u Chart

Objective: Monitor defects per unit when inspection-unit size varies.

Methodology: Compute \(\bar{u}\) and set limits for each subgroup of size \(n_j\):

\[ \text{UCL}_u = \bar{u} + 3\sqrt{\frac{\bar{u}}{n_j}}, \qquad \text{LCL}_u = \bar{u} - 3\sqrt{\frac{\bar{u}}{n_j}}. \]

Phase 3: Acceptance Sampling

Experiment 8: Single Sampling Plan (OC Curve, Producer's and Consumer's Risks, AOQ, AOQL, ATI)

Objective: For a given single sampling plan \((n, c)\), construct the OC curve and compute AOQ, AOQL, and ATI.

Methodology: Compute \(P_a\) for a range of \(p\) values using the Poisson approximation. Plot the OC curve. Compute AOQ = \(P_a \cdot p\) and identify AOQL. Compute ATI = \(n + (1-P_a)(N-n)\). Identify producer's risk at AQL and consumer's risk at LTPD.

Experiment 9: Determination of Single Sampling Plan by (a) Lot Quality Approach and (b) Average Quality Approach

Objective: Design a single sampling plan that meets specified risk requirements.

Methodology (a): Given AQL, \(\alpha\), LTPD, \(\beta\) — compute the ratio \(R = \text{LTPD}/\text{AQL}\) and determine \((n, c)\) from standard tables.

Methodology (b): Given AOQL and lot size \(N\) — determine \((n, c)\) using Dodge–Romig-type constants \(y_c\).

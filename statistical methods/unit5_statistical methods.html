Unit 5 — Theory of Attributes

Notations, class & ultimate frequencies, consistency of data (2 & 3 attributes), independence, association of attributes, Yule's coefficients of association & colligation.

Topics Covered

Notations Order of Class Ultimate Frequencies Consistency Independence Association Yule's Q Colligation

1. Notation for Attributes

DEFINITION

An attribute is a qualitative characteristic of an individual that cannot be measured quantitatively but can only be classified (e.g., honesty, gender, eye colour, smoking habit).

If a population is classified by attribute A, an individual either possesses A (denoted \(A\)) or does not possess A (denoted \(\alpha\), the "negation"). Similarly we use \(B, \beta\), \(C, \gamma\), etc.

Frequency Notation

  • \(N\) = total number of individuals.
  • \((A)\) = number possessing attribute A; \((\alpha)\) = number not possessing A.
  • \((AB)\) = number possessing both A and B; \((A\beta)\) = number with A but not B; etc.

2. Order of Class Frequencies

Class frequencies are classified by order = number of attributes specified.

  • Zero-order: only \(N\) — no attribute specified.
  • First-order: \((A), (\alpha), (B), (\beta)\) — one attribute specified.
  • Second-order: \((AB), (A\beta), (\alpha B), (\alpha\beta)\) — two attributes specified.
  • n-th order: all \(n\) attributes specified (one of \(2^n\) combinations).

3. Ultimate Class Frequencies

The frequencies of the highest-order classes (where every attribute is specified as either present or absent) are called ultimate class frequencies. For 2 attributes there are \(2^2 = 4\) ultimate frequencies; for 3 attributes there are \(2^3 = 8\).

Lower-order frequencies can be expressed as sums of ultimate frequencies. E.g.

\[ (A) = (AB) + (A\beta), \quad (B) = (AB) + (\alpha B), \quad N = (AB)+(A\beta)+(\alpha B)+(\alpha\beta). \]

4. Consistency of Data

DEFINITION

A set of class frequencies is consistent if it does not violate the basic axiom that every frequency must be non-negative.

4.1 Conditions of Consistency for Two Attributes

The four ultimate frequencies must all be \(\ge 0\). Since

\[ (AB) \ge 0,\; (A\beta) = (A) - (AB) \ge 0,\; (\alpha B) = (B) - (AB) \ge 0,\; (\alpha\beta) = N - (A) - (B) + (AB) \ge 0, \]

the consistency conditions reduce to:

\[ 0 \le (AB) \le \min\{(A), (B)\}, \qquad (AB) \ge (A) + (B) - N. \]

4.2 Conditions for Three Attributes

For 3 attributes, the consistency conditions are more numerous (8 ultimate frequencies must all be \(\ge 0\)). Common ones include:

  • \((ABC) \ge 0\)
  • \((AB) \ge (ABC),\; (AC) \ge (ABC),\; (BC) \ge (ABC)\)
  • \((A\beta\gamma), (\alpha B\gamma), (\alpha \beta C), (\alpha\beta\gamma) \ge 0\) — leads to inequalities on all 8 ultimate frequencies.
EXAMPLE 1

Given: \(N = 1000,\; (A) = 600,\; (B) = 500,\; (AB) = 200\). Check consistency.

\((A\beta) = 600 - 200 = 400 \ge 0\) ✓.

\((\alpha B) = 500 - 200 = 300 \ge 0\) ✓.

\((\alpha\beta) = 1000 - 600 - 500 + 200 = 100 \ge 0\) ✓.

Data are consistent.

EXAMPLE 2 (Inconsistent)

\(N = 100,\; (A) = 70,\; (B) = 60,\; (AB) = 20\). Check.

\((\alpha\beta) = 100 - 70 - 60 + 20 = -10 < 0\) ✗ — inconsistent.

5. Independence of Attributes

DEFINITION

Two attributes \(A\) and \(B\) are independent if the proportion of \(A\) in the population is the same as the proportion of \(A\) within \(B\):

\[ \dfrac{(AB)}{(B)} \;=\; \dfrac{(A)}{N}. \]

Equivalently:

\[ (AB) \;=\; \dfrac{(A)(B)}{N}. \]

Comparison with the Expected Frequency

  • If \((AB) > (A)(B)/N\): \(A\) and \(B\) are positively associated.
  • If \((AB) < (A)(B)/N\): \(A\) and \(B\) are negatively associated (disassociation).
  • If \((AB) = (A)(B)/N\): \(A\) and \(B\) are independent.
EXAMPLE 1

Among 1000 people: 200 have attribute \(A\), 300 have \(B\), and 60 have both.
Expected \((AB)\) under independence = \(200 \cdot 300 / 1000 = 60\). Observed = 60.
\(A\) and \(B\) are independent.

EXAMPLE 2

Same population, but \((AB) = 80\). Expected = 60, so \(80 > 60\) ⇒ positive association.

6. Association of Attributes — Yule's Coefficient \(Q\)

DEFINITION

Yule's coefficient of association:

\[ Q \;=\; \dfrac{(AB)(\alpha\beta) - (A\beta)(\alpha B)}{(AB)(\alpha\beta) + (A\beta)(\alpha B)}. \]

Properties of \(Q\)

  1. \(-1 \le Q \le 1\).
  2. \(Q = 0\) ⇔ attributes are independent.
  3. \(Q = +1\) ⇒ perfect positive association ((Aβ) = 0 or (αB) = 0).
  4. \(Q = -1\) ⇒ perfect negative association ((AB) = 0 or (αβ) = 0).
  5. Sign of \(Q\) reflects direction of association.

7. Coefficient of Colligation \(\omega\)

DEFINITION

Yule's coefficient of colligation:

\[ \omega \;=\; \dfrac{\sqrt{(AB)(\alpha\beta)} - \sqrt{(A\beta)(\alpha B)}}{\sqrt{(AB)(\alpha\beta)} + \sqrt{(A\beta)(\alpha B)}}. \]

Relation between \(Q\) and \(\omega\)

\[ Q \;=\; \dfrac{2\,\omega}{1 + \omega^2}. \]
  • \(-1 \le \omega \le 1\); \(\omega\) and \(Q\) have the same sign.
  • \(|\omega| \le |Q|\) (colligation is more conservative).
  • Both equal 0 when attributes are independent.

8. Worked Examples

EXAMPLE 1

Among 200 people: \((AB) = 80,\; (A\beta) = 30,\; (\alpha B) = 40,\; (\alpha\beta) = 50\).

\(Q = (80 \cdot 50 - 30 \cdot 40)/(80 \cdot 50 + 30 \cdot 40) = (4000 - 1200)/(4000 + 1200) = 2800/5200 = 0.538\).

Strong positive association.

\(\omega = (\sqrt{4000} - \sqrt{1200})/(\sqrt{4000} + \sqrt{1200}) = (63.25 - 34.64)/(63.25 + 34.64) = 28.61/97.89 = 0.292\).

Verify: \(2\omega/(1+\omega^2) = 0.584/(1.0853) ≈ 0.538\) ✓.

EXAMPLE 2 (Independence)

Among 1000 individuals: \((AB) = 60, (A\beta) = 140, (\alpha B) = 240, (\alpha\beta) = 560\).

Check: \((A) = 60+140 = 200,\; (B) = 60+240 = 300\); expected \((AB) = 200 \cdot 300 / 1000 = 60\) — matches observed.

\(Q = (60 \cdot 560 - 140 \cdot 240)/(60 \cdot 560 + 140 \cdot 240) = (33600 - 33600)/(33600 + 33600) = 0\). Attributes are independent.

Key Take-aways

  • Attributes are qualitative; classified by presence (capital) or absence (Greek).
  • Class frequencies have orders 0, 1, 2, …, n; ultimate frequencies are the highest-order ones.
  • Consistency: every ultimate frequency must be ≥ 0.
  • Independence: \((AB) = (A)(B)/N\).
  • Yule's \(Q \in [-1, 1]\) measures association; \(Q = 0\) ⇔ independence.
  • Colligation \(\omega\) related by \(Q = 2\omega/(1 + \omega^2)\).

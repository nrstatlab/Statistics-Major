1. Data Entry, Formatting & Worksheet Management

Excel Structure

  • Workbook: the entire .xlsx file.
  • Worksheet: a single tabbed sheet within the workbook.
  • Cell: intersection of a column (A, B, …) and row (1, 2, …); referenced as A1, B5, etc.
  • Range: a rectangular block of cells written A1:C10.
  • Formula: starts with =, e.g., =A1+B1.

Data Entry Tips

  • Use Tab to move right, Enter to move down.
  • For dates: type 12/05/2026 or use =TODAY().
  • For text starting with a number (e.g., a code "007"): prefix with apostrophe '007.
  • Ctrl+; inserts today's date; Ctrl+Shift+; inserts time.

Formatting

  • Number formats — General, Number, Currency (₹), Percentage, Date, Scientific.
  • Font, colour, borders, cell shading via the Home tab.
  • Merge & Center — combine adjacent cells.
  • Wrap Text — multi-line content within a cell.
  • Format Painter — copy format from one cell to another.

Worksheet Management

  • Right-click a tab → Insert/Delete/Rename/Move/Copy.
  • Freeze Panes (View → Freeze Panes) to keep row/column headers visible while scrolling.
  • Split window for side-by-side comparison of distant rows.
  • Save as: .xlsx (default), .csv (text), .pdf.

2. Arithmetic Functions

FunctionPurposeExample
=SUM(range)Add all values=SUM(A1:A10)
=AVERAGE(range)Mean=AVERAGE(A1:A10)
=PRODUCT(range)Multiply all values=PRODUCT(A1:A5)
=QUOTIENT(num, denom)Integer part of division=QUOTIENT(17, 5) → 3
=MOD(num, divisor)Remainder=MOD(17, 5) → 2
=ROUND(num, digits)Round to N decimals=ROUND(3.14159, 2) → 3.14
=POWER(num, exp)numexp=POWER(2, 10) → 1024
=SQRT(num)Square root=SQRT(64) → 8
=ABS(num)Absolute value=ABS(-7) → 7
EXAMPLE 1

Suppose A1:A5 contains 10, 20, 30, 40, 50. Then =SUM(A1:A5) = 150, =AVERAGE(A1:A5) = 30, =PRODUCT(A1:A5) = 12 000 000.

EXAMPLE 2

Check whether 84 is divisible by 7: =MOD(84, 7) → 0 ✓. Number of complete dozens in 100 eggs: =QUOTIENT(100, 12) → 8.

3. Logical Functions

FunctionPurposeExample
=IF(test, val_if_true, val_if_false)Conditional return=IF(A1>35,"Pass","Fail")
=AND(cond1, cond2, …)TRUE if all conditions are TRUE=AND(A1>0, A1<100)
=OR(cond1, cond2, …)TRUE if any condition is TRUE=OR(A1="M", A1="F")
=NOT(cond)Inverts TRUE/FALSE=NOT(A1=0)
=IFERROR(formula, alt)Show alt when formula errors=IFERROR(A1/B1,"div by 0")
=IFS(c1,v1,c2,v2,…)Multiple if-then chains=IFS(A1>90,"A",A1>75,"B",TRUE,"C")
EXAMPLE 1 (Pass/Fail)

A class mark in A2; formula in B2: =IF(A2>=35, "Pass", "Fail"). Drag down for all students.

EXAMPLE 2 (Grade)

Compute grade using nested IF:

=IF(A2>=90, "A+", IF(A2>=75, "A", IF(A2>=60, "B", IF(A2>=35, "C", "F"))))

Or with IFS: =IFS(A2>=90,"A+", A2>=75,"A", A2>=60,"B", A2>=35,"C", TRUE,"F").

4. Lookup & Reference Functions

VLOOKUP (Vertical Lookup)

=VLOOKUP(lookup_value, table_array, col_index, [range_lookup])
  • lookup_value — what to find.
  • table_array — table where the first column has lookup keys.
  • col_index — which column to return (1-based).
  • range_lookup — FALSE for exact, TRUE for approximate.

HLOOKUP (Horizontal Lookup)

Same idea but searches the first row instead of the first column.

XLOOKUP (Modern, in Excel 365 / 2021)

=XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode])

Replaces VLOOKUP / HLOOKUP with cleaner syntax; can lookup left of the key column.

INDEX / MATCH (powerful combo)

=INDEX(return_range, MATCH(lookup_value, lookup_range, 0))
EXAMPLE 1 (VLOOKUP)

Student table: A2:C10 has ID, Name, Marks. To get marks of student with ID 105: =VLOOKUP(105, A2:C10, 3, FALSE).

EXAMPLE 2 (XLOOKUP)

Same task with XLOOKUP: =XLOOKUP(105, A2:A10, C2:C10, "Not found"). Cleaner and more flexible.

5. Cell Referencing

TypeSyntaxBehaviour when copied
RelativeA1Adjusts both row & column
Absolute$A$1Stays fixed
Mixed (row absolute)A$1Column adjusts, row fixed
Mixed (column absolute)$A1Column fixed, row adjusts

Keyboard: press F4 in the formula bar to toggle through A1 → $A$1 → A$1 → $A1.

EXAMPLE 1

If B2 = =A2*$D$1 (with D1 = tax rate), copying down to B3, B4, … gives =A3*$D$1, =A4*$D$1, … — the tax rate stays fixed.

EXAMPLE 2 (Multiplication Table)

Build a 10×10 multiplication table. In B2 type =$A2 * B$1 and drag right and down. The column-fixed $A2 and row-fixed B$1 ensure each cell multiplies its row label by its column label.

6. Data Management

6.1 Sorting

Data tab → Sort & Filter → Sort. Choose column, order (A→Z or smallest to largest), optional secondary keys.

6.2 Filtering

Data → Filter (turns on drop-down arrows in column headers). Click an arrow to show only rows that match selected values, or apply text/number/date filters.

6.3 Conditional Formatting

Home → Conditional Formatting. Highlight cells that satisfy rules:

  • Greater/Less than a value
  • Top/Bottom N items
  • Data bars, colour scales, icon sets
  • Custom formula: e.g., highlight entire row where column D = "Failed".

6.4 Data Validation

Data → Data Validation. Restrict what users can type:

  • Whole number / decimal between min and max.
  • List → drop-down with allowed values (comma-separated or a range).
  • Date / Time ranges.
  • Custom formula validation.
  • Input message and Error Alert tabs let you set helpful prompts.
EXAMPLE 1 (Drop-down)

For column "Gender", select cells → Data Validation → Allow: List → Source: Male, Female, Other. Now those cells will only accept those values.

EXAMPLE 2 (Whole number range)

For column "Marks", restrict to whole numbers between 0 and 100. Any out-of-range input will trigger an error pop-up.

7. Excel Add-Ins for Analysis

7.1 Analysis ToolPak

DEFINITION

The Analysis ToolPak is a built-in add-in providing one-click tools for descriptive statistics, regression, ANOVA, t-test, F-test, histogram, sampling and more.

To enable: File → Options → Add-Ins → Manage: Excel Add-Ins → Go → tick "Analysis ToolPak" → OK. The "Data Analysis" button now appears under the Data tab.

Tools available

  • ANOVA: Single Factor, Two-Factor with/without Replication.
  • Correlation, Covariance.
  • Descriptive Statistics.
  • Exponential Smoothing.
  • F-Test Two-Sample for Variances.
  • Fourier Analysis.
  • Histogram.
  • Moving Average.
  • Random Number Generation.
  • Rank and Percentile.
  • Regression.
  • Sampling.
  • t-Test (Paired, Two-Sample Equal/Unequal Variances).
  • z-Test: Two Sample for Means.

7.2 Solver

DEFINITION

Solver is an add-in for optimisation problems — find the values of variable cells that maximise / minimise a target cell subject to constraints.

To enable: File → Options → Add-Ins → Manage: Excel Add-Ins → Go → tick "Solver Add-in" → OK. Appears under Data tab.

Solver Set-up

  • Set Objective: target cell (max, min or value of).
  • By Changing Variable Cells: the decision variables.
  • Subject to Constraints: inequalities or equalities.
  • Solving methods: Simplex LP (linear), GRG Non-linear, Evolutionary.
EXAMPLE 1 (Descriptive Stats)

Data in A2:A50 (marks of 49 students). Data → Data Analysis → Descriptive Statistics → Input A2:A50 → tick "Summary statistics" → output to a new cell. Excel prints Mean, Median, Mode, SD, Variance, Skewness, Kurtosis, Range, Min, Max, Sum, Count.

EXAMPLE 2 (Solver LP)

Maximise profit \(Z = 4x + 5y\) subject to \(x + y \le 8,\; 2x + y \le 10,\; x, y \ge 0\).

Set up cells for \(x, y\); a cell for Z = =4*X + 5*Y; two constraint cells. Solver: Max Z, by changing X & Y, subject to the constraints, method = Simplex LP. Click Solve → optimum \(x = 0, y = 8, Z = 40\) (the corner \((0,8)\) satisfies both \(x+y\le 8\) and \(2x+y\le 10\) and beats \((2,6)\) where \(Z = 38\)).

Key Take-aways

  • Cells, ranges and formulas are the building blocks.
  • Arithmetic functions: SUM, AVERAGE, PRODUCT, QUOTIENT, MOD, POWER, SQRT.
  • Logical: IF, AND, OR, NOT, IFERROR, IFS.
  • Lookup: VLOOKUP / HLOOKUP / XLOOKUP / INDEX-MATCH.
  • References: relative (A1), absolute ($A$1), mixed; F4 toggles.
  • Data management: sort, filter, conditional formatting, data validation.
  • Analysis ToolPak adds ANOVA, regression, t-test, descriptive stats; Solver adds optimisation.

What Does "Fitting a Distribution" Mean?

PROCEDURE
  1. Compute observed frequencies \(f_x\) of \(x = 0, 1, 2, \ldots\) and total \(N = \sum f_x\).
  2. Estimate the parameters from the data (e.g. \(\bar x\) for Poisson, \(\bar x / n\) for Binomial).
  3. Calculate theoretical probabilities \(p(x)\).
  4. Find expected frequencies \(\hat f_x = N \cdot p(x)\).
  5. Compare observed and expected frequencies (often via χ²-test).

Experiment 1 — Fitting Binomial (Direct Method)

PROBLEM

Eight coins are tossed and the number of heads \(X\) is recorded; the experiment is repeated 256 times.

Fit a Binomial distribution.

Solution:

\(N = 256\); \(\sum f x = 0 + 6 + 60 + 156 + 268 + 280 + 192 + 70 + 8 = 1040\). Mean \(\bar x = 1040/256 = 4.0625\).

For Binomial \(B(n, p)\): \(\bar x = np\). Here \(n = 8\), so \(\hat p = 4.0625/8 = 0.5078\), \(q = 0.4922\).

Experiment 2 — Fitting Binomial (Recurrence Method)

Use the same data. With \(\hat p = 0.5078,\; q = 0.4922\), \(p/q = 1.0317\).

\[ P(X = x + 1) = \dfrac{n - x}{x + 1} \cdot \dfrac{p}{q} \cdot P(X = x). \]

Start with \(P(0) = q^8 = 0.4922^8 = 0.00355\). Then:

  • \(P(1) = (8/1)(1.0317)(0.00355) = 0.0293\)
  • \(P(2) = (7/2)(1.0317)(0.0293) = 0.1058\)
  • \(P(3) = (6/3)(1.0317)(0.1058) = 0.2183\)
  • \(P(4) = (5/4)(1.0317)(0.2183) = 0.2816\)
  • ...continue similarly to verify against direct method.

The recurrence avoids repeated factorial computation.

Fit a Poisson distribution.

Solution: \(N = 100\); \(\sum f x = 0 + 38 + 24 + 9 + 4 + 0 = 75\). Mean \(\bar x = 0.75 = \hat\lambda\).

\(P(X=x) = e^{-0.75}(0.75)^x / x!;\; e^{-0.75} = 0.4724\).

Experiment 4 — Fitting Poisson (Recurrence Method)

Same data; \(\hat\lambda = 0.75\). Recurrence:

\[ P(X = x + 1) = \dfrac{\lambda}{x + 1} P(X = x), \qquad P(X=0) = e^{-0.75} = 0.4724. \]
  • \(P(1) = (0.75/1)(0.4724) = 0.3543\)
  • \(P(2) = (0.75/2)(0.3543) = 0.1329\)
  • \(P(3) = (0.75/3)(0.1329) = 0.0332\)
  • \(P(4) = (0.75/4)(0.0332) = 0.0062\)

Matches direct method ✓.

Experiment 5 — Fitting Negative Binomial (Direct Method)

PROBLEM

Number of policies a salesman has to make to close 3 deals (using \(X\) = failures before 3rd success). Frequencies recorded over 200 weeks:

x01234567+
f1027404534221210

Solution: Estimate parameters by method of moments. Compute sample mean and variance, equate:

Mean \(\hat \mu = rq/p\) and Variance \(\hat \sigma^2 = rq/p^2\). Solve: \(\hat p = \mu/\sigma^2\), \(\hat r = \mu^2/(\sigma^2 - \mu)\).

(Numerical mean ≈ 3.36, variance ≈ 3.30; here variance < mean indicates Poisson is better. In a true NB sample variance > mean.)

Once \(\hat r,\hat p,\hat q\) are known, compute \(P(X=x) = \binom{x+r-1}{x} p^r q^x\) and expected = \(N \cdot P(x)\).

Experiment 6 — Fitting Negative Binomial (Recurrence Method)

\[ P(X = x + 1) = \dfrac{(r + x) q}{x + 1} P(X = x), \qquad P(X = 0) = p^r. \]

For estimated \(\hat r = 3, \hat p = 0.5\) say: \(P(0) = 0.125;\; P(1) = (3 \cdot 0.5/1)(0.125) = 0.1875;\; P(2) = (4 \cdot 0.5/2)(0.1875) = 0.1875\), and so on. Then expected = \(N \cdot P(x)\).

Experiment 7 — Fitting Geometric (Direct Method)

PROBLEM

Number of failures before the first success of a marketing call:

x012345+
f4024141075

\(N = 100;\; \sum f x = 0 + 24 + 28 + 30 + 28 + 25 = 135\). Mean \(\bar x = 1.35\).

For Geometric: mean = \(q/p\) ⇒ \(q = p \bar x\) and \(p + q = 1\) gives \(p = 1/(1 + \bar x) = 1/2.35 = 0.4255\); \(q = 0.5745\).

Experiment 8 — Fitting Geometric (Recurrence Method)

From \(P(X=0) = p\), recurrence is simply

\[ P(X = x + 1) = q \cdot P(X = x). \]

For \(p = 0.4255,\; q = 0.5745\): \(P(0)=0.4255,\; P(1) = 0.2444,\; P(2) = 0.1404,\; P(3) = 0.0807, \ldots\) — matches.

Experiment 9 — Fitting Hypergeometric Distribution

PROBLEM

Lots of size 50 contain 10 defective items. From each lot, samples of 5 items are drawn (without replacement) and number of defectives is observed. The frequencies over 500 lots are:

x012345

Solution: \(N = 50, M = 10, n = 5\) (parameters known from problem context).

\(P(X = x) = \dfrac{\binom{10}{x}\binom{40}{5-x}}{\binom{50}{5}} = \dfrac{\binom{10}{x}\binom{40}{5-x}}{2 118 760}\).
